Tune linearization

Linearization tuning adjusts the level of brightness in the darkest part of the image. Channel values from three images are averaged to calculate the black level offset values. These offset values are then subtracted from the pixel output to obtain the corrected value.

  1. On the Pipeline tab, click Linearization, and then click a region in the Regions group.
  2. (Optional) Deselect the Bls Enable check box you would like to use default black level values for black level subtraction (BLS).
  3. Calculate linearization values:
    1. Click a region in the Regions group.
    2. Click Load Image, select the applicable black level reference image, and click Open.
      The image values are listed in the table fields with the average values listed underneath.
    3. Click Recalculate to recalculate linearization values for the R/G and G/ B values.
    4. Click OK.
  4. Repeat process for each region.
  5. Click File > Save to Project to save any changes made.
