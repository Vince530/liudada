Tune MCTF

Motion compensated temporal filter (MCTF) futher reduces noise in images.

Tune ANR and TF and load presets as needed.
  1. On the Pipeline tab, click TF and then click a region in the Regions group.
  2. Click Load MCC Images and load at least 20 MCC images.
  3. Click Load Flat Image and load at least 20 flat images.
  4. Click Base Functions and tune after the 20th frame from DC64 to DC4 (Full pass is not used for MCTF).
    1. Set the thresholds (Ytrsh, Utrsh, and Vtrsh) to be smaller than the SAD of POIs for moving objects.
    2. Set the thresholds (Ytrsh, Utrsh, and Vtrsh) to be larger than the SAD of POIs for stationary objects.
    3. Click Save to apply all changes.
  5. Click Blend Factor.
    1. Set a2 to make moving and stationary objects have a similar noise level.
    2. Set a2 according to the filter strength (FS) (e.g., if FS is high, set a2 as high).
