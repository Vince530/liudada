Test charts and equipment required for capturing images

Table : Test charts for capturing images
Chart or equipment Applicable tuning procedures


Macbeth color checker (MCC)

  • CCM
  • Linearization
  • ABF noise profiling
  • ANR noise profiling
  • Bayer AWB tuning


18% gray chart
  • AWB reference point
  • AWB low-light LUT


Diffuser filter
  • LSC tuning
  • Linearization


ISO12233 resolution chart

  • ASF sharpness tuning
Table : Equipment for capturing images
Equipment Purpose


Light booth

Provides the required types of light sources:

  • Daylight – D75, D65, D50
  • Fluorescent – CW, TL84
  • Incandescent – A and H


Light meter
  • Measures lighting intensity
  • Must be able to show the lighting intensity in lux


Neutral density (ND) filter
  • Used when the light booth cannot control lighting intensity
