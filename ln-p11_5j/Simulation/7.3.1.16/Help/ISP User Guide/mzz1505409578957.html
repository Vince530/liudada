GIC parameters

The following table lists the parameters used for tuning the GIC module. The default values for the parameters are suitable for most situations but may require adjustment in some cases to achieve your specific objectives.

Noise Std Lut

Table that plots the local noise intensity (in terms of standard deviation) at 65 points; known as the noise profile
  • The table is computed by the Chromatix tool. Generally, the noise profile does not need tuning. Compute for each sensor.
  • Higher values result in weaker GIC and stronger PNR
 Range = [0, 512.0] Advanced tuning – Use the computed values; if it is necessary to tune this parameter, request guidance from the CE team
