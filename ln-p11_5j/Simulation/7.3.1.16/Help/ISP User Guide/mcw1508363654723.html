Capture dot chart images

Related tuning block: ICA1

The light can be D65, TL84, or D55 and should be between 200 and 400 lux.

  1. Place camera on the tripod and position the camera parallel and aligned (not rotated) with the dot chart.
  2. Set the camera to the minimum gain to reduce noise and set the exposure so there is no flickering from the light source.
  3. Set the chart to 100%+ FOV. The image should include only the chart.
  4. Ensure the captured image has no motion blur.
