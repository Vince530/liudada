Run Parser Command

Execute the following steps to use the Run Parser Command feature:

  1. Load a project.
  2. From the File menu, select Binary File/Parser > Set Parameter Parser Path to set the path for the parameter parser.


    Ensure that the parameter parser path points to the Windows version of the parameter parser tool that comes along with the source code.
  3. From the File menu, select Binary File/Parser > Run Parser Command.


    The Run button is enabled if all the requirements are met. Missing or invalid parameters disable it.
