Tune CS

Use this procedure to set the must tune and often tune CS parameters.
Note: Consider the following tips when tuning CS with other color-related modules:
  • Tune all other color processing modules first and complete color correction before suppressing chroma tints with CS.
  • While tuning other color processing modules, set all y_weight_lut values to 1 to disable CS.
  • To suppress strong color saturation, tune the color correction matrix, gamma, and color noise reduction (ANR) before tuning CS.
  • Before removing color tints with CS, first tune color noise reduction (ANR).
  1. If needed, adjust Knee Point Lut to define flexible chroma suppression based on luma intensity.
    1. In the Simulation tab, select IPE > CS.
    2. Click the appropriate Region.
    3. Select Cs 20 Rgn > Knee Point Lut Tab > Knee Point Lut.
    4. Adjust the knee point values, as needed.
  2. If needed, adjust Y Weight LUT Tab to adjust the suppression weight based on intensity.
    1. In the Simulation tab, select IPE > CS.
    2. Click the appropriate Region.
    3. Select Cs 20 Rgn > Y Weight Lut Tab > Y Weight Lut.
    4. Adjust the luma weight values, as needed.
  3. If needed, adjust C Thr 1 LUT Tab to adjust the first chroma threshold.
    1. In the Simulation tab, select IPE > CS.
    2. Click the appropriate Region.
    3. Select Cs 20 Rgn > C Thr 1 Lut Tab > C Thr 1 Lut.
    4. Adjust the threshold values, as needed.
  4. If needed, adjust C Thr 2 LUT Tab to adjust the second chroma threshold.
    1. In the Simulation tab, select IPE > CS.
    2. Click the appropriate Region.
    3. Select Cs 20 Rgn > C Thr 2 Lut Tab > C Thr 2 Lut.
    4. Adjust the threshold values, as needed.
