CS parameters

The following table lists the parameters used for tuning the CS module. The default values for the parameters are suitable for most situations but may require adjustment in some cases to achieve your specific objectives. Tune each parameter for the applicable regions.
Table :   Chroma suppression parameters
Parameter Name Description Tuning Factory Default
Knee Point LUT Sixteen luma knee points that the user can control. This LUT applies to all three LUT indexes (y_weight_lut, c_thr1_lut, c_thr2_lut). Define sixteen knee points that can be useful for tuning Y weight and chroma weight. Evenly-spaced points: 0 17 34 51 68 85 102 119 136 153 170 187 204 221 238 256
Y Weight LUT The weight given to chroma suppression at the associated luma knee point.

In general, apply a strong suppression for dark and highlight areas and do not touch a middle tone color.

Define a Y weight to associate with each of the sixteen luma knee points.

Range: [0.0, 1.0]

A higher value = weaker chroma suppression

A lower value = stronger chroma suppression

If all sixteen values are 1.0, it is the same as disabling the chroma suppression module.

 All 0.0 (zero)
Chroma Threshold 1 LUT Set exposure control triggers to be based on Gain or Lux Index. A chroma value less than this threshold is suppressed with Y weight.

A chroma value between the thresholds is suppressed with an interpolated Y weight.

A higher value = stronger chroma suppression

A lower value = weaker chroma suppression

Must be less than chroma threshold 2.

Range: [0, 128]

 All 0 (zero)
Chroma Threshold 2 LUT Enable flag for Gr/Gb cross-channel processing. A chroma value greater than this threshold is not suppressed.

A chroma value between the thresholds is suppressed with an interpolated chroma weight.

A higher value = stronger chroma suppression

A lower value = weaker chroma suppression

Must be greater than chroma threshold 1.

Range: [0, 128]

 All 2
