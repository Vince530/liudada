GRA parameters

The following table lists the parameters used for tuning the GRA module. The default values for the parameters are suitable for most situations but may require adjustment in some cases to achieve your specific objectives.
Table :   Basic GRA tuning parameters
Parameter name Description Tuning Factory default Tuning required
Grain Strength

Number of value bits to generate for grain.

For example, for value of 3, the grain noise added is in the range from -7.

Adjust as needed to control grain.

Higher value = more grain

Lower value = less grain
