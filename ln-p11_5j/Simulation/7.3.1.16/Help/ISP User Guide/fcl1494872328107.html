Simulation Tools

The simulation tools allow you to apply the effect of a tuned project on an image to see how the tuned parameters affect the image.

  • Run AWB and pipeline simulations from the simulation tab
  • Run a pipeline simulation with JPEG input
  • Run a JPEG simulation from the command line
  • Run an AWB simulation from the command line
  • Run AWB simulation from the AWB tab
  • Run a pipeline simulation with raw input from the command line
  • Run a simulation using Command Line Assistant
