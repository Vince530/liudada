Run AWB simulation from the AWB tab

An alternative method to run an AWB simulation is to use the AWB tab. This method allows for input of multiple images to get a more accurate AWB simulation.

Perform AWB tuning and generate new binary for the target. Get these two DLL files from the target and place them in a local folder together:
  • com.qti.stats.awb.dll
  • com.qti.stats.awbsimulationwrapper.dll
When asked for the DLL file, give the path to com.qti.stats.awb.dll. The simulator finds the other DLL file in the same folder.
  1. Click the Single Image or Multi Image button.
  2. When the file dialog opens, a message in the top left corner prompts for the AWB Sim dll file. Navigate to the applicable file in the target build. Click Open.
  3. The file dialog opens again and prompts for the tuned project .bin file. Navigate to the applicable file in the target. Click Open. NOTE: For subsequent simulations, it is important to update the .bin file after making any tuning parameter updates.
  4. The file dialog opens again and prompts for a JPEG file to use as input for the simulation. If you clicked the Multi Image button, select multiple JPEG files. Navigate to and select the input files.
    The AWB simulator runs immediately after the input files are selected.
  5. When the simulation is complete, a message box opens with the results of the simulation and the location of the CSV file that contains the resulting R, G, B, gain and CCT values.
  6. Select Help > Chromatix log folder to view information about the AWB simulation that can be helpful for debugging.
