Measure the lux level with a light meter

When capturing images for camera tuning, the lux level of each image must be recorded. Perform the following steps to properly measure the lux level.

  1. Set the light meter to Ev mode where the meter shows the intensity of the light that falls on its sensor (white bulb).
  2. Place the light meter on the chart and ensure that the light meter sensor is facing the camera. The following example reads 520 lux.
