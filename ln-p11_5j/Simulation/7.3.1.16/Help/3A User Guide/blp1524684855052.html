Fine-tune AWB initial tuning parameters

Use this procedure to update the initial AWB gain values after initial tuning.
  1. Click the AWB tab.
  2. To tune the initial gain and initial CCT parameters, do the following:
    1. From the List View tab, select Initial Gain and CCT.


    2. Tune the initial gain values and initial CCT, as needed.
  3. To tune the BG stats config mode, do the following:
    1. From the Tree View tab, select Initial Gain and CCT (V1) > BGStat Conf > BGStat Mode.


    2. In the right pane, select the appropriate BGStat Mode value from the list.


  4. Click File > Save Project.
