Stats screening parameters

To view the stats screening parameters, open a project, click the AWB tab, and click Stats screening in the List View tab.

Dark Pixel Threshold: Core Data Defines three darkness threshold values for R, G, and B, respectively. Represents the lowest acceptable brightness value. Specify applicable threshold for each trigger row.

If stats brightness value is lower than this threshold, stats are ignored.

Range: [0.0 - 999.9]
