PDAF: Test AF speed

  1. Enable and capture log.
  2. Run the following adb commands: 
    adb root
adb remount
adb shell setprop persist.camera.stats.debug.mask 48
  3. Put the device on a tripod and focus on a far object.
  4. Bring in a chart and wait for focus, then remove the chart and wait for focus on the far object again.
  5. Search for ALGO Complete in the logs. You should see something like this example: 
    ALGO Complete finalpos 92 totaltime: 330 fine_search_time 265 focus_converge_time 65

    In the log sample above, the total latency time is 330 ms (total latency = focus converge + fine search).

  6. Verify that the AF speed varies when the object distance changes between each focus and that they are within the expected range. If speed does not meet expectations, adjust the applicable PDAF parameters.
