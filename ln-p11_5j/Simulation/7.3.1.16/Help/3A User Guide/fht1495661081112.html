Tune FV_min

A lower FV_min includes more weak edges but it may decrease the contrast of the FV curve, and increase the fail rate.

  1. Enable AF logging through ADB command.
  2. Edit the header file and set the FV_min value to zero and push the new library to the device.
  3. Place a diffusor over the sensor cover (to remove all texture from the scene).
  4. Turn off all the lights and capture logs.
  5. Calculate the number of pixels in the ROI. FV_min = FV value in the full-sweep search* / number of pixel in ROI.
  6. Repeat steps 2 through 5 for bright light conditions.
  7. Replace the FV_min in the header with the calculated values.
    Note: Because hardware and software stats are logged differently, the FV from hardware stats and the FV from software stats could be different for the same scene.
