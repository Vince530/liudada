Generalized scene analyzer parameters

To view the SA parameters, open a project, select the AWB tab, and click an SA name in the Scene Analyzer section.

Coordinates of three vertices to define the detection zone.

  • For triangle shape, all 3 entries are used.

Table :   AWB SA confidence level tuning parameters
Parameter Name Description Tuning Factory Default Must Tune?
Confidence Level Name Text name of SA confidence level n/a Level 1-6 Assigned by tool
Confidence Level Compute Type

Method used to compute a final confidence level by merging all the confidence levels

  • Multiply - Multiply all confidence levels
  • Weighted Average - Apply Confidence Weight to each confidence level, then average
  • Minimum - Select minimum confidence level
  • Maximum - Select maximum confidence level
 Select preferred method Multiply Yes
Confidence Weight Weight to apply to specified confidence level. Only used if the Confidence Compute Type is Weighted Average. Use default unless using Confidence Compute Type = Weighted Average 1.0 Use default or modify
