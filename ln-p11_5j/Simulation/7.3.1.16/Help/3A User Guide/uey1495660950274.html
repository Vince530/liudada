Exposure table

The camera image sensor amplifies pixel information using an analog amplifier to create sensor gain. The image sensor can also be exposed to light for a longer time, so each pixel is allowed to collect more light. Sensor gain and pixel exposure are then controlled by an exposure algorithm to achieve constant brightness of a frame, object, or scene with respect to a target value.

Using a longer exposure time (a lower gain) gives better SNR to the image, but causes a lower frame rate and blurriness due to motiona and handshake. Using a higher gain causes a higher frame rate, but the image has more noise.

If the image is under-exposed, it is too dark. If it is overexposed, the image is too bright, and colors can become saturated and washed out.

Figure : (1) Original image; (2) Under exposed; (3) Over exposed; (4) Target exposure

An exposure table is constructed to define the best choices for the trade off of exposure. An exposure table comprises exposure knee points, where each knee point has the following values:
  • Exposure Index – Specifies the exposure sensitivity for a given knee point
  • Gain – Specifies the amplification used for a given exposure
  • Exposure Time – Specifies the sensor integration time for a given exposure sensitivity
  • Increment Priority – Specifies the interpolation of exposure settings between two exposure knee points; can be incremented by either gain or exposure time first


A general practice is to maintain a higher frame rate whenever possible to keep the image sharp with less noise. The increment method usually favors exposure time by incrementing the gain first. However, under darker conditions, where noise becomes an image quality issue, gain is favored by increasing the exposure sensitivity using a longer exposure time.

Table : Exposure table parameters
Parameter Description
expIndex Defines the exposure index for the exposure knee point; autogenerated from gain and exposure time count
gain Defines exposure linear gain for exposure knee points
expTime Defines exposure time in nanoseconds for exposure knee points
incrementPriority Defines exposure increment priority from the knee point (e.g., gain, exp time); default value is Exp Time
sensitivityCorrectionFactor Define the binning factor for the normalization of exposure sensitivity between exposure tables
minPreviewVertBlankTime Defines minimum sensor veritical blanking time for preview mode
