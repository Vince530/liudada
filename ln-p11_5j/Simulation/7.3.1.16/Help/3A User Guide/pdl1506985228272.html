Fine-tune stats screening

Use this procedure to tune dark pixel threshold triggers and to tune saturation pixel threshold triggers.

Tune or add dark pixel threshold triggers

  1. Open the project and click the AWB tab.
  2. On the List View tab, select Stats Screening.
  3. Find the Dark Pixel Threshold section.

  4. To edit the lux index trigger range, in the first column, double-click the entry in brackets. Update the lux index start and end values and click outside the box to save the change.
  5. To edit the red/green/blue dark pixel threshold data, in the second column, double-click the entry in brackets. Update the red/green/blue data values for this trigger and click outside the box to save the change.
  6. To add more trigger levels, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous three steps to set the trigger range and associated core data.
  7. Click File > Save To Project to save the changes.

Tune or add saturation pixel threshold triggers

  1. Open the project and click the AWB tab.
  2. On the List View tab, select Stats Screening.
  3. Find the Saturation Pixel Threshold section.

  4. To edit the lux index trigger range, in the first column, double-click the entry in brackets. Update the lux index start and end values and click outside the box to save the change.
  5. To edit the red/green/blue dark pixel threshold data, in the second column, double-click the entry in brackets. Update the red/green/blue data values for this trigger and click outside the box to save the change.
  6. To add more trigger levels, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous three steps to set the trigger range and associated core data.
  7. Click File > Save To Project to save the changes.
