Misleading color zone parameters

To view the MLC parameters, open a project, select the AWB tab, and click Misleading Color Zone in the List View tab.

Table :   AWB detection zone tuning parameters
Parameter name Description Tuning Factory default Tuning required
Detection Zones Text name of MLC detection zone N/A Zone 1 to 10 Use default value
Zone Description

Text description for the detection zone

Max 128 characters

 Use a name that is descriptive of the detection zone. Initial set named by tool during AWB initial tuning Use predefined value or custom name
AWB Zone Group

Identification number of the group with which this detection zone is associated

Recommend use value of 4 predefined detection zone groups. Use a sequential number for additional groups. 0 to 9 Use predefined value
AWB Zone Type Defines the shape of the zone as rectangle, triangle, or CCT Recommend use value of 4 predefined AWB zone groups. Select appropriate value for new zones. Rectangle Recommend take predefined value
Trigger Level Defines trigger associated with detection zone Only lux level is supported for detection zones Lux level Recommend take predefined value
Detection zone trigger data Start and end lux level trigger values for a specified detection zone Recommend use a common set of lux level triggers for most AWB processes Initial set computed by tool during AWB initial tuning Recommend take predefined value
Detection zone vertices core data Coordinates of three vertices to define the detection zone. All 3 entries are used for triangle shape. First two entries are used for rectangle shape. First entry is used for CCT. Recommend use value of 4 predefined detection zone groups. Adjust predefined values or create new zones as needed. Initial set computed by tool during AWB initial tuning Recommend take predefined value
Table :   AWB MLC zone tuning parameters
Parameter Name Description Tuning Factory Default Must Tune?
Misleading Color Zones Text name of MLC zone N/A Zone 1 to 10 Use default value
Zone Description

Text description for the MLC zone

Max 128 characters

 Use a name that is descriptive of the MLC zone. Initial set named by tool during AWB initial tuning Use predefined value or custom name
AWB Zone Group

Identification number of the group with which this MLC zone is associated

Enter zone number for related detection zone. 0 to 9 Recommend take predefined value
AWB Zone Type Defines the shape of the MLC zone as rectangle, triangle, or CCT Recommend use value of 10 predefined MLC zones. Rectangle Recommend take predefined value
1st and 2nd Trigger Levels Defines type of data used for MLC zone triggers

Options available for trigger levels 1 and 2 are:

  • Lux Index
  • CCT
  • Valid Stats Cnt Ratio
  • Agw Rg
  • Agw Bg
  • Zone group # Cnt Ratio
 Initial set computed by tool during AWB initial tuning Recommend take predefined value
3rd Trigger Level Defines type of data used for 3rd MLC zone trigger

Third level is only for flash.

If the MLC zone should not be used for certain flash sensitivity (K2) values, set up this trigger as appropriate. See the section AWB flash concept.

Flash Sensitivity Off Use default.
MLC zone trigger data Start and end trigger values for a specified MLC zone Recommend use value of 10 predefined MLC zones. Initial set computed by tool during AWB initial tuning Recommend take predefined value
MLC zone core data

Coordinates of three vertices to define the MLC zone.

  • For triangle shape, all three entries are used.
  • For rectangle shape, first two entries are used.
  • For CCT, only the first entry is used for the start/end values.
 Recommend use value of 10 predefined MLC zones. Initial set computed by tool during AWB initial tuning Recommend take predefined value
