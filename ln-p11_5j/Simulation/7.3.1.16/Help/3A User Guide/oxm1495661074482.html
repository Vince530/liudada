Initial tuning

The initial AF tuning involves configuring, testing, and tuning the parameters in the 3 A > AF > Chromatix Contrast AF > Single Search structure that are used by the continuous autofocus (CAF), touch autofocus (TAF), and single AF algorithms. CAF and TAF monitor the scene for scene changes within the defined region of interest (ROI). If the algorithm detects a change and the scene change criteria is met, then it triggers the single AF algorithm to search for the appropriate lens position.

These parameters are instrumental in determining search regions, lens positions, and help ensure the following:

  • Actuator moves the lens in the expected range
  • Lens movement is damping and settling according to requirements
  • Focus value stats are generated correctly
  • Single search AF parameters
  • Load AF preset XMLs
  • Test actuator driver
