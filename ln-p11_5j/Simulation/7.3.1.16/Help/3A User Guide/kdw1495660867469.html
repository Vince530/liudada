Run HAF: TOF test

This CAF Scan test checks the quality distance in millimeters returned by the laser sensor driver.

Setup

Repeat the test for multiple distances like 10 cm, 20 cm, and 100 cm from the device on a tripod with 500+ lux light condition.

Analysis and corrective action

If graph looks like Graph A, then the distance returned by the sensor is correct for the test distances. If the graph looks like Graph B, then the distance returned by sensor has a huge deviation from the expectation. Contact the vendor to determine why the library is returning the wrong distance output. In most cases, the issue is because the cover glass that is applied on top of the laser sensor needs calibration. If the calibration is incorrect, then it is likely to output inconsistent distances.

Note: Qualcomm does not modify an external vendor’s library.
