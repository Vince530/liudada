Dynamic skip frames parameters

To view the dynamic skip frames parameters, open a project, click the AWB tab, and click Dynamic Skip Frames in the List View tab.

Table : Dynamic skip frames tuning parameters
Parameter name Description Tuning Factory default
Enable Flag

Enable or disable dynamic skip frames module

 Always set to 1 (enable) 1
Trigger Each trigger row defines the start and end values of the difference ratio zone.

The ratio is the difference between the previous and current AWB decision. Both R/G ratio and B/G ratio are computed.

The maximum difference ratio (R/G ratio vs B/G ratio) is compared to the trigger zones to determine the number of frames to skip.

 Set up trigger zones by specifying start and end points for each zone.
