DCIAF: Test AF speed

  1. Enable and capture AF log.
  2. Put the device on a tripod and focus on a far object.
  3. Bring in a chart and wait for focus.
  4. Then remove the chart and wait for focus on the far object again.
  5. Search for ALGO Complete in the logs. You should see something like this: 
    ALGO Complete finalpos 92 totaltime: 330 fine_search_time 265 focus_converge_time 65

    The total latency time is 330 ms (total latency = focus converge + fine search).

  6. Verify that the AF speed varies when the object distance changes between each focus. Verify that the different speeds are within the expected range. If speed does not meet expectations, adjust the applicable DCIAF parameters.
