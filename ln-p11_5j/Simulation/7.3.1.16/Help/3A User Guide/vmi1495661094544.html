Tune extreme color detection and luma adjustment

Extreme color detection and luma adjustment compensate for images that contain a large amount of a particular bold color. This prevents overexposure by applying Lens Shading Correction (LSC), linearization, and white balance to adjust the region of influence based on extreme color.

Tune the AWB Reference Points for D65.
Each region is verified to determine whether it is outside the R/G, B/G, and R/B combinations. This compensation prevents extreme colored scenes from being overexposed or saturated. Luma weighting and frame luma biasing can be adjusted for extreme red, blue, and green regions.
Figure : (L) Extreme color luma biasing; (R) Normal metering

The extreme color detection feature is tuned using the generic multi-zone tuning guidelines. Default AEC triggers are initially tuned in a common step and then copied over.

  1. In the Parameter Editor, navigate to 3A > AEC > Metering > Mtr Luma Target > Extrm Clr Luma Target.
  2. To enable extreme color luma target adjustment, select the Value check box.
  3. Adjust the following Extrm Clr Luma Target parameters by entering the new value in the appropriate field.
    • Theshold/Interpolation Percentage Red: Defines threshold of RG ratio as extreme red; interpolation percentage (interpPct) extends from the threshold for stable transition in detection.
    • Theshold/Interpolation Percentage Blue: Defines threshold of BG ratio as extreme blue; interpPct extends from the threshold for stable transition in detection.
    • Theshold/Interpolation Percentage Green B: Defines threshold of BG ratio as extreme green; interpPct extends from the threshold for stable transition in detection.
    • Theshold/Interpolation Percentage Green R: Defines threshold of RG ratio as extreme green; interpPct extends from the threshold for stable transition in detection.
    • Theshold/Interpolation Percentage Green Center: Defines threshold of RG and BG ratio jointly as extreme green; interpPct extends from the threshold for stable transition in detection.
    • Low Threshold for region Luma: Defines the effect of the color stats by luma value of stats point. Effect is discounted if the luma value is below the threshold.
    • High Threshold for region Luma: Defines the effect of the color stats by luma value of stats point. Effect is discounted if the luma value is below the threshold.
    • Threshold Color Stats Percentage: A range in terms of percentage of color stats to determine strength of luma target adjustment.
    • Extrm Clr Luma Target Zone: Defines the luma target adjustment scaler for each detected extreme color. A smaller value means a lower luma target.
