Actuator settle time

This test checks if the lens settles once the lens movement command is issued by checking the focus value consistency at the same lens position for multiple frames.

Setup

Place the ISO 12233 resolution chart 30 cm from the device on a tripod with a 500+ lux light condition.

Analysis and corrective action

If the graph looks like Graph A, then the damping/ringing settings are configured correctly. If the graph looks like Graph B, then either of the following conditions exist:

  • The damping/ringing settings are wrong
  • There is a timing issue between the stats parsing in ISP and sending it to the AF core

Repeat the test two more times to confirm the problem. If you confirm the problem, then adjust the damping settings and rerun the test. If the problem still exists, then check the damping by using a laser in the lab.

If it is not a damping issue, then it could be a timing issue with the ISP. For example, if the fps is 30 fps, then AF expects the stats to arrive every 33 ms. A delay in the stats could cause the issue. Debug the issue with the engineer responsible for your ISP tuning.
