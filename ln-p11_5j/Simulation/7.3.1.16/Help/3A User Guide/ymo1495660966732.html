Follower logic for dual camera

If DCIAF is enabled, tune haf.reserve[48] through [90] to enable and configure the follower logic. This logic provides a master/slave relationship in the AF behavior of the cameras. Using this logic, the slave follows the master. The reserve parameters are tuning params that provide the flexibility to control some of follower's independent decisions.
