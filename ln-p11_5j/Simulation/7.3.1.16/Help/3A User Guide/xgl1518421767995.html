Fine tune AWB dual camera sync custom reference points

Prerequisite: Complete the initial AWB tuning process for each camera in its own Chromatix project. In the initial AWB tuning, measure the regular reference points of both cameras under the same light condition.

Fine tune custom reference points

  1. Open the project and click the AWB tab.
  2. In the Parameter Editor, navigate to 3A > AWB > Dual Cam Sync (V1) > Custom Ref Pt to view the existing reference points. Determine what additional reference points or value changes are needed.
  3. To add a new custom reference point, do the following:
    1. In the left pane, navigate to 3A > AWB > Dual Cam Sync (V1) > Custom Ref Pt and select Ref Point.
    2. In the right pane, click the plus sign (+) next to the row under which you want to add a custom reference point structure (for example, click + next to Ref Pt [0] to create Ref Pt [1]). Add as many reference point structures as required.
    3. In each new Ref Pt structure row, click the dots in the Value column to set the Condition, Cal Factor, and Point values. Repeat for each reference point structure you added.
  4. To update an existing custom reference point, navigate to 3A > AWB > Dual Cam Sync (V1) > Custom Ref Pt and select Ref Point[x]. Make adjustments to the Condition, Cal Factor, and Point values, as needed.
  5. To delete an existing custom reference point navigate to 3A > AWB > Dual Cam Sync (V1) > Custom Ref Pt > Ref Pt and click the minus sign (-) next to the existing reference point you want to delete.
  6. Click File > Save to project.
Note: Dual camera sync processing works by referring to the dualcamSyncV1.xml file in both cameras. If it is necessary for a custom reference point to be added to the dualcamSyncV1.xml file in both cameras, use the following sequence:
  1. Capture the gray patch with two cameras under the same light condition.
  2. Get the rg, bg ratio from the gray patch image for each camera. Refer to the sgw rg, bg.
  3. Add A(rg, bg) to dualcamSyncV1.xml in Camera I as custRefPt[x].
  4. Add B(rg, bg) to dualcamSyncV1.xml in Camera II as custRefPt[x].
    Note: The array order [x] must be the same in each camera.
