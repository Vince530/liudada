Fine-tune stats subsampling

Use this procedure to tune the stats subsampling mode and mask pattern.
  1. Click the AWB tab and then the Tree View tab.
  2. To change the subsampling mode, select Stats SubSample (V1) > Mode and select the appropriate mode value.
  3. To update the mask pattern, select Stats SubSample (V1) > Pattern
  4. Update the twelve pattern indexes to match the preferred mask. The values are: 1 = used, 0 = discard.


  5. Select File > Save Project
