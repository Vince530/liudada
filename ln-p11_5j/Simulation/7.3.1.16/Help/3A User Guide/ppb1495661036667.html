AF monitor

The main functionality in the AF monitor state is scene change detection. Intuitively, scene change means that the current scene is not in-focus, which causes the algorithm to trigger a search to find the best lens position for focus.

Each algorithm has its own criteria for detecting scene changes. For example, CAF relies on gyro parameters, sum of absolute differences (SAD) parameters, and SAD with respect to reference frames (SADR) parameters to detect a scene change. In comparison, PDAF relies on a combination of the defocus value of the current frame, the confidence of the algorithm, and the stability of the scene.

The AF monitor is also responsible for scene panning detection. To avoid unnecessary searches, the algorithms differentiates between scene changes and scene panning. The search algorithms do not start searching until the scene is stable, that is, not panning. In short, scene change and scene panning detection are the two fundamental functionalities in AF monitor state.
