Luma target and exposure index

The AEC algorithm controls sensor gain and exposure time to match the current luma with the target luma target. The current luma indicates the image brightness of the current frame and is calculated based on metering tuning and AE statistics that are derived before the gamma block. The luma target is set for each light condition.
