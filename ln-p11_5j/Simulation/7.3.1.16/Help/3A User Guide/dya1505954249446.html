Tune custom reference points

All custom reference points must be added using this procedure.
  1. In the left pane, click the AWB tab, then click Custom Reference Points in the list.
  2. In the Custom Reference Points panel, do one of the following:
    • Click on the words Zone 1 to set up the first custom reference point.
    • Click the last zone name, then click the plus sign (+) to add a subsequent custom reference point.
  3. In the RG-BG Point panel, select the Enable Flag check box and add the coordinates of the custom reference point in the RG: and BG: fields.
  4. To set a distance weight trigger, do the following in the Distance Weight Vector panel:
    1. In the Trigger column, double-click the {0, 0} entry to open the edit dialog. Update the start and end values and click outside the box to save the change.
    2. In the Data column, double-click the (0) entry to open the edit dialog. Update the weight value for this distance trigger and click outside the box to save the change.
    3. To add more triggers, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous two substeps to set the trigger range and associated weight value.

  5. To set a lux index weight adjustment, do the following in the Weight Adjustment panel:
    1. In the Trigger column, double-click the {0, 0} entry to open the edit dialog. Update the start and end values and click outside the box to save the change.
    2. In the Data column, double-click the (0) entry to open the edit dialog. Update the weight value for this distance trigger and click outside the box to save the change.
    3. To add more triggers, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous two substeps to set the trigger range and associated weight value.
  6. To adjust a trigger range or weight value, double-click the appropriate entry to open the edit dialog. Update the value and click outside the box to save the change.
  7. Click File > Save To Project to save changes.
