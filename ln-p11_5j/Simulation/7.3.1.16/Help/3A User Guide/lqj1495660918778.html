Optional basic AEC tuning procedures

The following AEC tuning procedures are basic, but optional. Completion of these optional procedures is recommended for optimal camera performance.

Table :   Optional basic AEC tuning procedures
Tuning procedure Description
Tune AE bias weight AE bias metering sets the spot weighted (small area) and center weighted (wider area) fields of detection for AE. AE detection can also be an average of the entire field.
Tune AWB gain for luma compensation (v5.1) The AWB gain for luma compensation feature helps to autocalibrate RBG response of different CCT and module-to-module variations.
Tune preview convergence Convergence determines the speed and stability of the image when the preview AE is changing.
Tune camcorder convergence parameters Camcorder convergence determines the speed and stability when camcorder scenes are changing.
Tune AEC camera startup parameters Use this procedure to achieve the fastest startup AEC convergence speed.
Tune touch AEC Touch AEC allows you to modify the luma metering weight by touching a specific region of the preview display screen.
Tune face AEC Face AEC allows the luma metering weight to be optimized to a detected face in the image on the preview display screen.
Tune ISO value ISO 100 gain is a measure of the sensitivity of the digital camera sensor.
Tune antibanding vs. ISO priority Use this procedure if your requirements make it necessary to prioritize a manual ISO over antibanding.
