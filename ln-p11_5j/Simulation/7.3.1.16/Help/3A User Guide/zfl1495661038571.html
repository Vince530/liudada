Enable 3A debug log messages (3A v5.x and v6.0)

Use adb commands to enable 3A debug logs for the specified chipset.

MSM8952/MSM8976/MSM8956/MSM8953/MSM8940/MSM8937/MSM8917/MSM8996/MSM8998/SDM660/SDM630

adb root
adb remount
adb shell setprop persist.camera.mobicat 2
adb shell setprop persist.camera.stats.debug <3A EXIF Debug Mask>
