Run CAF Scan tests

CAF Scan performs fundamental tests on basic AF functionality. Run the CAF Scan tests, analyze the results, and perform any necessary corrective action before doing any advanced tuning

At this stage of the tuning process, run the following CAF Scan tests.

  • Bi-directional full sweep
  • Closed loop actuator range
  • Open loop actuator range
  • Actuator settle time
  • Focus curve contrast and tail

  • Distance to lens mapping

  1. Open the Chromatix 7 camera tuning tool.
  2. Connect the device to a USB port on the computer that is running Chromatix.
  3. Open the device to the home screen. Do not open the camera app.
  4. In the Chromatix menu, select Tools> CAF Scan Test to open the CAF scan menu.
  5. Click Run CAF Scan. The CAFScan window appears.
  6. Select the test you want to run.
  7. Set up the device and environment as directed in the Instructions pane. The instructions include the test chart type, the required distance between the device and the chart, and the required brightness.
  8. When the environment is set up, click Run Test. The test launches the camera app and starts the focus action.
  9. After the focus is stable, click OK when the focus ring disappears and the tool displays the OK prompt. When the test finishes, the tool exits the camera app, parses the log, and plots the data on a graph in the Results pane.
