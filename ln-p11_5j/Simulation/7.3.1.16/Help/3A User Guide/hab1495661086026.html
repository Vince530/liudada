Focus value contrast and tail

This test checks the quality of the focus value graph by checking its contrast ratio (max value FV/ min value FV). A larger ratio indicates a good contrast ratio for the autofocus algorithm.

Setup

Place the ISO 12233 resolution chart 30 cm from the device on a tripod with a 500+ lux light condition.

Analysis and corrective action



If graph looks like Graph A, then the Bayer AF (BAF) configuration is correctly configured. If the graph looks like Graph B or Graph C, then either the BAF settings are wrong or there could be an error in the ISP parsing of BAF stats. Repeat the test two more times to confirm the problem.

Try to tune BAF with different settings like changing the kernel coefficients or tuning coring to see if the contrast ratio improves
