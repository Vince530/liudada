Tune fast or aggressive AEC parameters

Tune fast and aggressive AEC start up parameters for first frame, convergence, and arbitration. For more information about fast and aggressive AEC, see AEC camera startup.

First frame

  1. Fast AEC: To allow a larger exposure adjustment when there are saturated stats, in the Parameter Editor navigate to 3A > AEC > Start Up > Start Up Conv > Fast AEC > First Frame Bias.
    1. To enable Start Up Bias, select the Value check box.
    2. Adjust parameter values as needed.
  2. Aggressive AEC: To allow a larger exposure adjustment when there are saturated stats, in the Parameter Editor navigate to 3A > AEC > Start Up > Start Up Conv > Aggressive AEC > First Frame Bias.
    1. To enable Start Up Bias, select the Value check box.
    2. Adjust parameter values as needed.

Convergence

  1. Fast AEC: To control the algorithm convergence until it reaches its target or times out, in the Parameter Editor navigate to 3A > AEC > Start Up > Start Up Conv > Fast AEC.
    1. Adjust parameters as needed.
  2. Aggressive AEC: To control the algorithm convergence until it reaches its target or times out, in the Parameter Editor navigate to 3A > AEC > Start Up > Start Up Conv > Aggressive AEC.
    1. Adjust parameters as needed.

Arbitration

  1. To speed up convergence, use a custom exposure table that first uses gain before increasing the exposure time until the frame rate drops. In the Parameter Editor navigate to 3A > AEC > Start Up > Start Up Arb.
    1. Adjust parameters as needed.
