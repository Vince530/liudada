Tune generic startup parameters

If needed, use the camera startup parameters to achieve the fastest startup AEC convergence speed.

  1. In the Parameter Editor, navigate to 3A > AEC > Start Up > Start Up Conv > Generic.
  2. Click the name of the camera startup parameter to be adjusted and enter the appropriate value in the field. The following camera startup parameters can be adjusted:

    • Start Exp Index: Exposure index that AEC should have at camera startup.

    • Start Up Aggressiveness: AEC convergence speed at startup.
