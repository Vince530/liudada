Tune anti-banding vs ISO priority

Anti-banding is always the top priority so that no flickering compromises the image quality. This action is not recommended, but if your requirements make it necessary to prioritize a manual ISO over anti-banding, reset the priority as described below.
  1. In the Parameter Editor navigate to 3A > AEC > Arbitration > ISO Data.
  2. To enable anti-banding, select the Value check box next to Enable Anti Banding Vs ISO Prio.
  3. Verify the manual ISO value is consistent across difference scenes.
