Preflash tuning

Preflash tuning controls the way AEC converges once the preflash turns on and illuminates the scene. Several parameters are considered advanced and are not recommended to tune.

Parameter Description Tuning Default
frameSkipMethod Specifies which type of frame skip is desired.
  • SmoothNoSkip – Continuous smooth frame-to-frame adjustments
  • SkipOne – Use this option only if synchronization issues occur between sensor and stats
  • SkipSystemLatency – Specifies that the next exposure adjustment will take place only once stats are updated to reflect previous exposure adjustment; use if convergence speed is critical
 SmoothNoSkip
aggressiveness Increase this value to converge faster. Range: [0.0, 1.0] 1
settledFrameCountThres The number of extra frames AEC should wait before declaring settled; reduce if sensor sync is reliable and thoroughly tested. Range: [0, 3] 2
thresBrightLevelDiff Lower this value to make preflash convergence reach a darker exposure before declaring settled. Range: [5, 50] 30
thresBrightLevel Increase this value to make preflash convergence reach a darker exposure before declaring settled. Range: [180, 210] 190
thresBrightSatDiff Lower this value to make preflash convergence reach a darker exposure before declaring settled. Range: [0.03, 0.15] 0.1
thresBrightSat Increase this value to make preflash convergence reach a darker exposure before declaring settled. Range: [0.15, 0.3] 0
minLuma AEC stops adjusting exposure during preflash convergence if luma drops below this threshold. Range: [8, 25] 15
minLumaAdj Increase this parameter to make convergence faster. Decrease this parameter to make convergence reach the same exposure repeatedly. Range: [0.5, 1.0] 0.55
