Tune warm startup

Warm startup uses the previous camera session exposure settings to initialize the new camera session.

In the Parameter Editor navigate to 3A > AEC > Start Up > Start Up Conv > Warm Start Up.
  1. To enable warm startup, select the Value check box.
  2. Set the start exposure index range percent parameter, if needed.
