Stats subsample concept

An entire set of stats often requires time to compute and load. While AWB can estimate a subset of stats through a skip process that selects stats every 2 to 4 columns, this does not contribute to uniform sampling.

The stats subsample filter provides a versatile way to select stats using masks that can be configured. AWB can then compute and directly use the set of subsampled stats or compute and load the average.

Figure : (L) Example of an image with no subsampling applied; (R) example of an image in which a subsampling mask has been specified every fourth column

The stats subsampling module consists of a mode parameter (skip stats or binning stats) and a pattern parameter. The subsampling pattern uses a 4x3 mask for selecting stats to process, with 4096 possible combinations. The following are examples of a 6-stat selection pattern (1 = use stat, 0 = discard stat):



When skip mode is selected, the subsampling process picks the stats in the cells marked with 1. For example, the figure on the left shows the 6-stat skip pattern. The figure on the right shows the skip mode result.



When binning mode is selected, the subsampling process picks the stats in the cells marked with 1 and averages them. For example, the first figure shows a 4-stat binning pattern. The second figure shows the binning result, which generates 256 stats in the 64x48 stats example.



On 64x48 grid stats, the pattern does not overlap; each stat is processed one time only.
