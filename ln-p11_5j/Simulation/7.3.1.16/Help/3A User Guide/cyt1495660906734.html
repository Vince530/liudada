Fast convergence

Preview convergence tuning modifies the image speed and stability when preview AE is changing. Fast convergence is recommended for preview mode.

To tune for fast convergence:

  1. In the Parameter Editor tab, navigate to 3A > AEC > Convergence > Fast Conv.
  2. Enter the appropriate values for Tol Mult Unit Step Size, Frame Skip Method, and Aggressiveness .
Parameter name Description Tuning Default
tolMultUnitStepSize Exposure index tolerance is tolMultUnitStepSize * unitStepSize; exposure index range within which AEC will not adjust Lower value = more precise convergence 2
frameSkipMethod Specifies which type of frame skip is preferred:
  • SmoothNoSkip for smooth convergence
  • SkipOne if there are synchronization concerns
  • SkipSystemLatency if speed is preferred to smoothness
  • SmoothNoSkip for continuous smooth frame-to-frame adjustments
  • SkipOne for in case of synchronization issues between sensor and stats only
  • SkipSystemLatency so next exposure adjustment will take place only once stats are updated to reflect previous exposure adjustment. Use this option if convergence speed is critical
Smooth No Skip
aggressiveness Controls convergence speed. Higher value = faster convergence 0.8
