Slow convergence

Slow convergence is recommended for video capture modes that require steady exposure and smooth adjustments.

In the Parameter Editor, navigate to 3A > AEC > Convergence > Slow Conv.

Parameter name Description Tuning Default
Basic tuning
tolMultUnitStepSize Exposure index tolerance is tolMultUnitStepSize * unitStepSize; exposure index range within which AEC will not adjust. Lower value = more precise convergence 2
frameSkipMethod Specifies which type of frame skip is preferred:
  • SmoothNoSkip for smooth convergence
  • SkipOne if there are synchronization concerns
  • SmoothNoSkip for continuous smooth frame-to-frame adjustments
  • SkipOne in case of synchronization issues between sensor and stats only
 Smooth No Skip
convSpeed Controls convergence speed. Higher value = faster convergence 2
stepFactorWhenFrameSkipEnabled Enable if frameSkipMethod is set to SkipOne to speed up convergence. Higher value = faster convergence 0.8
refFrameRate It is recommended to tune slow convergence using one reference frame rate (typically 30  fps). Enter the reference frame rate as a reference parameter so other frame rate mode tuning can be scaled.   30
Holding time – Length of time to hold off making exposure adjustments in the presence of motion (measured by the gyroscope) or scene disturbances (measured by luma changes)
enableHoldTime Enables holding off exposure adjustments when motion and exposure changes are small. Enable to enable Disabled
maxHoldTime Maximum amount of time AEC will hold off exposure adjustments when large disturbance is measured. Larger value = longer hold time 1
enableGyroBasedHT Enable using gyro to determine amount of disturbance. Enable to enable Disabled
holdTimeEnterDelay The amount of time AEC needs to be stable before holding time gets activated. Lower value = increase frequency of holding time 3
tolAdjFactor Hold time is not allowed if exposure index absolute delta from target is larger than tolerance times this threshold. Higher value = more frequent holding time 2
thresAbsExpDelta Range within which hold time is adjusted based on exposure deviation from target. Lower value = shorter hold time [13.7, 18]
expDeltaBasedHTAdjFactor Exposure-based hold time adjustment factor range, start needs to be larger than end. Lower value = shorter hold time [1.0, 0.7]
thresGyroMagnitude Range within which hold time is adjusted based on gyro magnitude. Lower value = shorter hold time [0.5, 4]
gyroBasedHTAdjFactor Gyro-based hold time adjustment factor range, start needs to be larger than end. Lower value = shorter hold time [1, 0.4]
Advanced tuning – Fine-tune convergence speed (not recommended to tune)
stepRegular Range of step sizes for slow convergence when there are no dark or bright regions. Final step size is interpolated based on convSpeed parameter.   [3, 5]
stepDark Range of step sizes for slow convergence when exposure is too dark and there are dark regions. Final step size is interpolated based on convSpeed parameter.   [10, 22]
stepBright Range of step sizes for slow convergence when exposure is too bright and there are bright regions. Final step size is interpolated based on convSpeed parameter.   [8, 16]
thresSlowDownAbsExpIndexDeltaDark Range of absolute exposure delta from target whithin which algorithm step size will be decreased to smoothly converge to target. Final step size is interpolated based on convSpeed parameter. Larger value = slower approach [17.3, 7.55]
thresSlowDownAbsExpIndexDeltaBright Range of absolute exposure delta from target whithin which algorithm step size will be decreased to smoothly converge to target. Final step size is interpolated based on convSpeed parameter Larger value = slower approach [7.55, 4.73]
maxRawStepAdj Adjust step size based on skip and frame rate. The step size is adjusted by a factor. Decrease/Increase this parameter to speed up convergence for frame rates faster/slower than the reference frame rate, respectively Larger value = faster convergence 2
thresNumBrightRgn The minimal ratio of bright regions required for bright step size to be used during convergence. Lower value = increase sensitivity 0.05
thresNumDarkRgn The minimal ratio of dark regions required for dark step size to be used during convergence. Lower value = increase sensitivity 0.05
