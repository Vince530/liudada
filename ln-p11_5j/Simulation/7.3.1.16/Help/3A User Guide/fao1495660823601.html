*[To Update] Tune flat scene detection

Flat scene detection should be disabled and Histogram AEC should be used instead for flat scene compensation.

  1. On the AEC Fine Tuning tab, click Flat Region. 
  2. Click and load an MCC raw image taken under TL84 lighting conditions.
    After the flat region value is calculated, a check mark displays next to Load Image. This indicates that the value was saved to the header file.
