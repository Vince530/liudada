Custom reference point parameters for AWB dual camera sync

Reference points tuning is very important for transforming the master decision point and the master GCM center point to that of the slave camera. If the transformation using regular reference points needs tuning, you can add custom reference points to correctly transform the master decision point and the master GCM center point to the slave decision point and the slave GCM center point, respectively.

Custom reference point processing behaves as follows:
  • If the condition of a custom reference point is AllCondition, the custom reference point is enabled at all brightness conditions
  • If the condition of a custom reference point is Outdoor, the custom reference point is only enabled when the current luxIdx is less than Lux threshold
  • If the condition of a custom reference point is Indoor, the custom reference point is only enabled when the current Lux Index is same or bigger than Lux threshold
  • If current Lux Index is in the allowed LuxRange, the process assumes that the scene has not changed. So the custom reference points set by the previous master are used to transform the master decision point and the master GCM center point to the slave decision point and the slave GCM center point.



The regular AWB reference points are usually measured using standard light sources of a standard light booth, such as SpectraLight III. If the outdoor GCM or FCM results with regular reference points are not accurate, it may be because the spectral response of the master camera or the slave camera for sunlight and standard light source (such as D75, D65, and D50) is not linear. In this case, set the Ref Pt Selection to UseCustRefPtOnly and create custom reference points as indicated in the following table.
