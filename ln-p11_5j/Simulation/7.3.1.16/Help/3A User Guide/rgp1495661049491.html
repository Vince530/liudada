Bi-directional full-sweep test

This test checks for lens module hysteresis. The test sweeps the lens from the far-end position to the near-end position and then sweeps again in the opposite direction. The peak of the focus value in both directions should be close to identical. If they show different peaks in opposite directions, then it means that there is a strong hysteresis in the lens module, which needs to be resolved.

Setup

Place the ISO 12233 resolution chart 30 cm from the device on a tripod with a 500+ lux light condition.

Analysis and corrective action



If the peak position in both directions is identical as in Graph A, then the lens module is good. If the peak positions differ by more than 10% as in Graph B, then there is a huge hysteresis in the module, which needs to be resolved.

Run the test two more times to confirm the difference. If confirmed, ask the module vendor if the module has acceptable hysteresis. If it is not acceptable, then get a good module from the vendor.
