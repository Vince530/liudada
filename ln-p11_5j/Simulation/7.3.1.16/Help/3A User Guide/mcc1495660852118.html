Prerequisites to AEC tuning

  1. Tune the following ISP modules, if they are applicable to the ISP project, using the procedures in the Qualcomm Spectra 2xx Camera ISP Tuning Guide :

    • Black level/Linearization

    • Lens shading correction

    • Gamma correction

  2. Disable the following advanced tuning parameters:
    • Luma adaptation
    • Bright region, Dark region, Extreme color detection and luma adjustment, and Histogram AEC luma adjustment
    • Flat scene detection
