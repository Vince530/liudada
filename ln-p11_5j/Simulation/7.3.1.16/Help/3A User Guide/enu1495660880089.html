Tune AEC lux index normalization

The AEC lux index normalization feature can reduce the effort for updating a new trigger point for each mode.

Tune, at a minimum, one exposure table. It is recommended to tune the preview mode exposure table.
AEC uses the exposure table for adjusting shutter speed and gain. The index of the exposure table (AWB Exp Index/Lux Index) is used as the operation trigger point for AWB, AF, or ISP modules. But if the exposure table is different for each camera mode, the exposure index or lux index may be different even in the same test condition, so a different operation trigger point is needed for AWB, AF, and ISP modules in each camera mode.

Tuning this feature sets a reference table and sensitivity that is used to calculate a normalized lux index and AWB exposure index by the reference setting. Using this feature, even if the exposure table is different in each camera mode, the output lux index and AWB exposure index will be the same in the same test condition.
