Value monitor overview

The value monitor instantiates the configuration and decision logic for each monitor algorithm. Although the monitor logic is similar to previous 3A versions, the structure, configuration, and operation of the value monitor is an entirely new implementation of that logic. The primary benefits of value monitors are flexible configurations and simplified tuning.

The following image displays the two main constructs for the value monitor.

  • Monitor algo config – Contains the sensitivity settings for each type of stat used in the CAF, PDAF, TOF, and DCIAF monitor algorithms .
  • Value monitor bank – Contains three sets of parameters (low, medium, and high) for each type of stat.

How a sensitivity parameter interacts with the table of parameters 

The figure below illustrates how a sensitivity parameter interacts with the table of parameters in the value monitor bank.

  1. In this example, the value for the gyro sensitivity is 50. This is the default value for all sensitivity parameters. The range for all sensitivity parameters is 1 to 100 (0 is off). All sensitivities between 1-100 are valid and are expected to result in a reasonable monitor behavior.
  2. Because the sensitivity is 50, the medium sensitivity parameters are selected. If the sensitivity parameter is 1, then the low sensitivity parameters are selected. If sensitivity is 100, then the high sensitivity parameters are selected. If the sensitivity is between 1 and 50 or 50 and 100, then linear interpolation between the low and medium or medium and high are used respectively.

    The parameters in each profile (low, med, high) are identical. However, their values are preconfigured to provide for varying levels of sensitivity based on the sensitivity parameter in the monitor algo config.

  3. CAF and any other monitor algorithms configured with the value monoitor set type of gyro are instantiated based on the configuration produced as a result of 1 and 2.

If a tuning engineer wants to make the gyro sensitivity more sensitive than the medium sensitivity profile, then the simplest way is to modify the gyro sensitivity to be more than 50. If the tuning engineer wants to make the gyro sensitivity less sensitive than the medium sensitivity profile, then modify the sensitivity to be less than 50.

NOTE: No parameters in the value monitor bank are changed while using the logical sensitivity parameters during the tuning process.
