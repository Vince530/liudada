Tune spotlight process parameters

After setting up the parameters that can detect a spotlight scene, tune the parameters that process the spotlight scene:

  1. Select Header type: 3A and navigate to AutoFocus > Tuning Algorithm > Continuous > AF Spotligh > t > AF Cont Spotlight Enable.
  2. Set the value of AF Cont Spotlight Enable to 1. This action forces the Hybrid AF (HAF) Phase Detection AF (PDAF) confidence indicator to False. As a result, when a spotlight scene is detected, the AF spotlight feature handles the scene. This can prevent an unreliable PD value during HAF converge and prevents the FV from ending in a false peak position for a spotlight flat scene.
  3. Use the guidelines for the AF spotlight process parameters to set the criteria for AF spotlight processing:

    • AF Cont Spotlight Far Flat Thresh

    • AF Cont Spotlight Macro Flat Thresh

    • AF Cont Spotlight Inc Flat Thresh

    • AF Cont Spotlight Dec Flat Thresh
