Value monitor decision logic

To detect and differentiate between scene changes and panning, a value monitor analyzes multiple input signals and then outputs binary decisions. A designated decision logic is applied to decide whether the current scene is panning or not. Therefore, a specific topology (decision logic) is used in the AF monitor. To simplify the design, the topology only takes binary inputs (0 or 1). Only decision logic is specified in topology.
