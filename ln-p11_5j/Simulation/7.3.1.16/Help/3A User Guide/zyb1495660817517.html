Enable AF core logging

Some situations require logging without enabling the full-sweep algorithm as described in the previous section. This section describes how to enable AF core logging without enabling the full-sweep algorithm.

  1. Use the following commands to enable AF core logging: 
    adb root
adb wait-for-device;
adb remount;
adb shell setprop persist.camera.stats.af.debug 5
  2. Start capturing the log before opening the camera. To capture the log, use the following command: 
    adb logcat –c
adb logcat –vtime > “filename”.log
    Note: Contact CE if you encounter difficulty enabling or locating the logs.
