AWB sensor calibration data

Camera software tuning data is based on the golden camera sensor module. But, sensor modules produced from the same manufacturing line have some variation in their characteristics. Knowing how an individual sensor varies from the golden sensor module allows the camera software to compensate for the variations. For AWB, the variations are especially important for R/B and B/G position.

The AWB sensor calibration module compares data from the current camera sensor module with data from the golden camera sensor module and makes the appropriate adjustments.



The AWB sensor calibration module computes the calibration factor for the current module as follows: current camera module calibration data / golden camera module calibration data

Refer to the Camera Module Selection and Calibration Data document for instructions on how to obtain the current camera module calibration data (numerator) from EEPROM\OTP data and obtain the golden camera module calibration data from the AWB header.
