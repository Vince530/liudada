Fine-tune decision aggregator

Prerequisite:
  • Before performing any fine tuning, be sure to run the AWB initial tuning procedure to generate initial parameter values.
  • Recommend fine-tune for other AWB modules (such as SA and Gain adjust) first, before tuning DA so that DA does not have more than three SAs that have non-zero confidence level in a given scene.
  • Consider various scenes when you tune DA, because tuning DA usually affects every scene.

Tune the confidence level adjustment factor

  1. Open the project and click the AWB tab.
  2. In the List View section, select Decision Aggregator.
  3. In the right panel find the Confidence adjustment based on confidence value section.

  4. In the Trigger column, double-click the {0,0} entry to open the edit dialog. Update the trigger start and end values and click outside the box to save the change.
  5. In the Data column, double-click the {0,0} entry to open the edit dialog. Update the adjustment factor and click outside the box to save the change.
  6. To add more triggers, use the Add above or Add below buttons to add a new blank trigger row. Repeat the two previous steps to set the trigger range and associated adjustment factor.
  7. To adjust a trigger range or data value, double-click the appropriate entry to open the edit dialog. Update the values and click outside the box to save the change.
  8. Click File > Save To Project to save the changes.
