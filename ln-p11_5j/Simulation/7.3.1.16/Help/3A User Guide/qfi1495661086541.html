Reduce saturation of faces with green foliage in the background

A higher face luma weight value can improve face brightness, but the side effect is that AE may become more unstable when detecting a face.

  1. Capture test images with 3A EXIF information and AEC log messages.
  2. Select File > Open Image and open the image in the Image Viewer.
  3. In the 3A Debug dialog box, click the Data tab and select AEC Debug Info > Luma and Luma Target Information.
  4. Check the Face luma parameter. A value of 0 indicates that face AE is disabled.
  5. Tune face AE.
  6. On the AEC Fine Tuning tab, click Edit Header.
  7. Select Header type: 3A and navigate to AEC > Face > Enable and verify that the value is 1 (enabled).

  8. If the issue persists when AE is enabled, add more weight for face metering (increase by 0.05 for each trial). Select Header type: 3A and navigate to AEC > Face > Weight
  9. In the lab, capture an image with a face and black background to use for verifying the results.
