Tune flash exposure tables

It is possible to use custom exposure tables for preflash and main flash.

After the LED turns on AEC must adjust quickly. For preflash, achieve good performance by using the same exposure table as for preview. However, a custom exposure table avoids dropping frame rate due to long exposure time and, therefore, converges faster.

For main flash, this is the exposure table for the actual snapshot. For preflash, good performance can be achieved by using the same exposure table as for snapshot without flash.

  1. In the Parameter Editor, navigate to 3A > AEC > Flash > Flash Arb
  2. Set the value for Exp Table ID Pre Flash.
  3. Set the value for Exp Table ID Main Flash
