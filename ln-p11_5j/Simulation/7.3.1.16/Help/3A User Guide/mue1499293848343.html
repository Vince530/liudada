Sensitivity correction factor

To maintain the same exposure sensitivity from one exposure table to another, sensitivity needs to be normalized by the sensor integration sensitivity that may differ due to pixel readouts, such as binning for each exposure table. If only one exposure table is used, there is no need to calculate this.

Full resolution sensor mode is the normal sensitivity since there is no binning sum factor. Therefore, the sensitivity correction factor is 1.

For quarter size or other sensor resolutions, the sensitivity correction factor (SCF) is related to the binning factor; it is also dependent on how the pixel values are read out. To get the SCF, refer to the sensor specification sheet or contact the sensor vendor for the binning sum factor for the sensor mode exposure table that will be used.
