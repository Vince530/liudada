LED smart ROI flash estimation

In AEC flash estimation, a smart ROI estimation is available to detect close objects within the scene and prevent saturation.

Smart ROI estimation allows detection of close objects within the scene. This process can be used to avoid a saturated foreground. The same estimation can be applied to flash AWB, in the sense that AWB can white-balance more on the front object with the ROI estimation algorithm. For example, in the pair of images below, without the smart ROI estimation, AEC overexposes the front object in the left image, whereas the image on the right is properly exposed with smart ROI estimation.

Figure : (L) Without smart ROI estimation; (R) With smart ROI estimation

To test smart flash, set up a scene with both background and foreground objects. The foreground object should be small, take up ~30% FOV, and receive and reflect light from the flash. The background object should be distant with less flash reflection. Before testing, disable the smart flash and take a reference image, ensuring the foreground object is saturated. Enable smart flash and compare the new image to the reference image.
