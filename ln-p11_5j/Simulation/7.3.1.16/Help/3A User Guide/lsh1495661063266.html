Considerations before tuning TOF

  • Gyro is essential to achieving the best TOF performance. If gyro is not available in the device, then we must rely on the sum of absolute difference (SAD) parameters for scene change detection and panning detection. This makes the triggering of TOF slightly slower.
  • The glass cover applied on top of the laser sensor must be of good quality material, must not be highly reflective, and must be dustproof. Without these characteristics, the laser sensor might report a false distance. In most cases, a crosstalk calibration is applied based on the glass material used to increase the signal strength.
  • For any depth-based algorithm, it is recommended to use a closed-loop actuator because it has less influence from gravity and hysteresis. Due to the high cost of a closed-loop actuator, some use an open-loop voice coil motor (VCM). If an open-loop VCM is used, we recommend using the lens sag compensation feature. To use lens sag compensation, ask the module vendor for the face down, face forward, and face up DAC values. It is recommended that each module has these calibrated values.
  • Current implementation of TOF AF algorithm does not support touch AF or face AF. If a touch or face is detected, then contrast AF is used instead of TOF AF.
