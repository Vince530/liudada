Run HAF: PDAF test

This CAF Scan test checks the quality phase difference (PD) value from the library along with the focus value quality. The two values should be in alignment.

Setup

Use the vertical stripes chart (see Appendix B and a 500+ lux light condition. Place the device in landscape orientation on a tripod and place the chart at various distances (10 cm, 20 cm, and 100 cm) from the device.

Analysis and corrective action

If the graph looks like the Graph A, then the PD library is outputting the correct phase difference. If it looks like Graph B, then contact the library vendor (e.g., Sony) to determine if PD calibration is at fault. If the library belongs to Qualcomm, contact CE engineer to report the issue.

Note: Qualcomm does not modify an external vendor’s library.
