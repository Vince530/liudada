Tune LED flash decision

Use this procedure to provide the main flash MWB gains and interpolation information required to combine multiple AWB decisions into a final decision.

Refer to Retrieve MWB target points for main flash.
  1. In the left pane, click the AWB tab, then click LED flash in the list.
  2. In the LED Flash window, click Flash Decision in the Tuning Module panel.
  3. In the Flash WB Point panel, select the appropriate LED Type.

    Note: If applying a calibration factor for single LED, select 'DualLED' as the LED Type.
  4. For single LED (without calibration factor), set up the Flash WB Point by entering the MAIN FLASH gains in the RG: and BG: fields.
  5. In the LED WB Point Interpolation panel, do the following to specify the ratio to use for interpolating the ambient AWB decision and the MWB decision with respect to K2:
    1. In the K2 column, double-click the {0, 0} entry to open the edit dialog. Update the MAIN FLASH K2 start and end values and click outside the box to save the change.

      Use the information gathered from the AWB log to determine how to fine-tune the K2 triggers. Refer to Retrieve K2 values for pre-flash and main flash.
    2. In the Data column, double-click the (0) entry to open the edit dialog. Update the LED point interpolation ratio value for this trigger zone and click outside the box to save the change.
    3. To add more triggers, use the Add above or Add below buttons to add a new blank flash sensitivity trigger row. Repeat the previous two steps to set the trigger range and associated interpolation ratio.
  6. In the Pre-flash LED WB Interpolation panel, do the following to specify the ratio to use for interpolating the pre-flash AWB decision with respect to lux index and valid stats count ratio:
    1. In the 1st Trigger column, double-click the {0, 0} entry to open the edit dialog. Update the lux index start and end values and click outside the box to save the change.
    2. In the 2nd Trigger column, double-click the {0, 0} entry to open the edit dialog. Update the valid stats count ratio start and end values and click outside the box to save the change.
    3. In the data column, double-click the (0) entry to open the edit dialog. Update the pre-flash LED interpolation ratio value for this trigger zone and click outside the box to save the change.
    4. To add more triggers, use the Add above or Add below buttons to add a new blank lux index or stats count ratio trigger row. Repeat the previous two steps to set the trigger range and associated interpolation ratio.
  7. To adjust a trigger range or data value, double-click the appropriate entry to open the edit dialog. Update the value and click outside the box to save the change.
  8. Click File > Save To Project.
