How the stable table and scene monitor tables work

The min_stable_cnt parameter in the stable table works with the defocus_threshold and depth_stable_threshold parameters in the scene monitor table to determine if a scene change is stable. Scene stability is one of the criteria for PDAF to exit the monitor state and to trigger a search.

The following example is labeled to explain the parameters:

  • If the defocus value (a) is larger than defocus_threshold, PDAF enters search state and resets the stable count.
  • If defocus value (b) is within range of depth_stable_threshold as compared with the defocus value of the previous frame, the stable count is incremented.
  • If stable count (c) is bigger than min_stable_cnt, PDAF triggers a search.
