FA SA parameters

To view the face-assist parameters, open a project, click the AWB tab, and click Face Assist AWB in the List View tab.

Table : Face-assist SA dark pixel/saturation pixel threshold tuning parameters
Parameter Name Description Tuning Factory Default
Dark pixel threshold: Trigger data Start and end lux index values for the trigger range. If lux index is in this range, face-assist uses the R/G/B core data as the dark threshold. Default values should be adequate.

Add multiple trigger ranges if needed.

Range: 0 - 999

 {0, 999}
Dark pixel threshold: Core data The R/G/B dark threshold for face stats. Represents lowest acceptable brightness value. Default values should be adequate.

Suggest to set a higher value for face stats than what is set for the Dark Pixel Threshold in the stats screening module.

If stats brightness value is lower than this threshold, stats are ignored.

Range: 0.0 - 15.0

 (5,5,5)
Saturation pixel threshold: Trigger data Start and end lux index values for the trigger range. If lux index is in this range, face-assist uses the R/G/B core data as the saturation threshold. Default values should be adequate.

Add multiple trigger ranges if needed.

Range: 0 - 999

 {0, 999}
Saturation pixel threshold: Core data The R/G/B saturation threshold for face stats. Represents highest acceptable brightness value. Default values should be adequate.

Suggest use the same value as regular Saturation Pixel Threshold in the stats screening module.

If stats brightness value is higher than this threshold, stats are ignored.

Range: 200.0 - 255.0

 (240,240,240)
