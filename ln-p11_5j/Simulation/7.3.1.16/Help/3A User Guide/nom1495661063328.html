Contrast AF (CAF)



Contrast AF (CAF) monitors the motion values from the gyro sensor and the sum of absolute differences (SAD) in luma values to detect scene changes and trigger a search. If the change meets the criteria for stability and confidence, then CAF triggers a search for the lens position that achieves the highest FV.

The default AF search algorithm is Single Search, which is a contrast-based algorithm.

The following are key points related to CAF and Single Search:

  • CAF contains the default monitor and search algorithms.
  • CAF monitor relies on changes in gyro values and SAD to detect scene changes.
  • CAF search is also called single AF and is a contrast-based algorithm.
  • Single Search searches for peaks in the FV to determine lens positions.
  • CAF is tuned before the HAF algorithms.
  • If any of the depth-based algorithms are enabled, CAF is only called as a fallback.
