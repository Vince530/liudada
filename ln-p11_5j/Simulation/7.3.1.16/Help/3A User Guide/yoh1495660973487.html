Tune CAF triggering

CAF monitors the motion values from the gyro sensor and the sum of absolute differences (SAD) in luma values to detect scene changes and trigger a search. CAF is responsible for the following:

  • Monitoring the scene to detect scene changes
  • Ensuring the scene is stable before triggering a search
  • Triggering a search

CAF is tuned by adjusting the sensitivity parameters located at 3 A > AF > Chromatix AFMontior > Monitor Algo Config > Topo CAFin the Parameter Editor. These parameters work with the parameters in the associated value monitor to configure the behavior of the algorithm.

The following table lists and describes each of the CAF sensitivity parameters. The range for each parameter is 0 to 100 (0 is off). The default is 50.

CAF sensitivity parameters

Parameter Name Descriptin Tuning
SADR Sensitivity value for SADR value monitor. Higher values, easier to trigger searches based on scene changes as determined by SADR.
Gyro Sensitivity value for gyro value monitor. Higher values, easier to trigger searches based on camera motion.
Gyro Panning Sensitivity value for gyro panning value monitor. Higher values, easier for a scene to be stable.
