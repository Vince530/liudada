Fine-tune misleading color zones

Prerequisite: Before performing any fine tuning, be sure to run the AWB initial tuning procedure to generate initial parameter values.

Use this procedure to fine-tune any of the predefined detection zones or MLC zones, or to create new detection or MLC zones.

Tune or add detection zones

  1. Open the project and click the AWB tab.
  2. In the List View tab, click Misleading Color Zone to open the tuning pane.
  3. In the Detection Zones section, select the detection zone you want to modify, or click the green plus sign to add a new detection zone.
  4. Type a descriptive name for the zone in the Zone Description field.
  5. Determine which group is applicable to this detection zone and type the number in the AWB Zone Group field.
  6. Select the appropriate zone type (rectangle, triangle, or CCT) from the AWB Zone Type list.
  7. Leave the 1st Trigger Level as Lux Idx. That is the only trigger available for detection zones.
  8. Adjust, add, or delete lux index ranges in the first column to set the preferred lux level triggers.
  9. Adjust or add entries to the second column to set the core data. The core data for detection zones represent the rectangle/triangle vertices or CCT range that defines the detection area enabled by the associated lux index trigger.
    • Rectangle – Set first two entries [(top-left vertex), (bottom-right vertex), (0,0)].
    • Triangle – Set all three entries to three vertices of the triangle.
    • CCT – Set the first entry to the CCT range [(lowCCT, highCCT), (0,0), (0,0)].

Tune or create MLC zones

  1. Open the project and click the AWB tab.
  2. In the List View tab, click Misleading Color Zone to open the tuning pane.
  3. In the Misleading Color Zones section, select the MLC zone you want to modify, or click the green plus sign to add a new MLC zone.
  4. Type a descriptive name for the zone in the Zone Description field.
  5. Determine which detection zone group is applicable to this MLC zone and type the number in the AWB Zone Group field.
  6. Select the appropriate zone type (rectangle, triangle, or CCT) from the AWB Zone Type list.
  7. Set the 1st Trigger Level to the preferred trigger type.
  8. Set the2nd Trigger Level to the preferred trigger type. If a second level trigger is not needed, set this to match the value of the 1st Trigger Level.
  9. Under the trigger level selection, there are four columns to configure the preferred set of trigger combinations and core data. Modify or add the preferred trigger ranges for the first trigger in the first column and for the second trigger in the second column. If you want to use the flash sensitivity trigger, set the third column values to (0,1).
  10. In the fourth column, adjust or add entries to set the core data. The core data for MLC zones represent the rectangle/triangle vertices or CCT range that defines the MLC area enabled by the corresponding trigger configuration.
    • Rectangle – Set first two entries [(top-left vertex), (bottom-right vertex), (0,0)].
    • Triangle – Set all three entries to three vertices of the triangle.
    • CCT – Set the first entry to the CCT range [(lowCCT, highCCT), (0,0), (0,0)].
