Tune the AEC default triggers (3A v5.x and v6.0)

Generate a common set of AEC zone triggers by capturing images under given lux levels in the light booth.

If possible, capture a JPEG image of a 100% FOV gray chart taken at 400 lux, D65 illuminant, containing 3A EXIF metadata.
Although each AEC feature has control of individual zones by setting its own number of zones and triggers, the initial tuning procedure for all AEC features is to generate a common set of triggers by taking pictures under given lux levels in the light booth. The Chromatix tool copies this common set of triggers to all the features that adopt the generic multi-zone-based tuning. Later, the triggers and number of zones can be changed individually for each feature at the fine tuning stage.

If you use auto tune for initial tuning of ISP modules, the auto tune process automatically computes the AEC default triggers from an image loaded in the Image Gallery.

For manual tuning or fine tuning, use the steps below to tune the AEC default triggers. This procedure is generally used during the fine tuning stage of image quality tuning.

  1. On the ISP Tuning tab, in the 3A Tuning group, click AEC.
  2. Click Tune next to Default Triggers.
  3. Enter the Lux index value of a JPEG image file taken of a 100% gray card under 400 lux lighting conditions using one of these methods:

    • Upload a JPEG image file taken of a 100% gray card under 400 lux and D65 illuminant that includes the 3A debug information in the EXIF header. Click and select the image file. The lux index is read from the EXIF header.

    • Manually enter the Lux index value from a photo of a 100% gray card image that was taken under 400 lux light condition.

  4. Update the Number of zones, if needed. By default, two triggers are generated resulting in three zones.


    The appropriate number of vertical boundary lines appears on the graph and the trigger Start Lux and End Lux values are calculated.
    Set “number of zones” = 3
“number of triggers” = “number of zones” - 1
The lux index for 400 lux light condition: “lux_idx_under_400”.
Trigger[0].start = lux_idx_under_400 - 100
Trigger[0].end = lux_idx_under_400 - 60
Trigger[1].start = lux_idx_under_400 + 90
Trigger[1].end = lux_idx_under_400 + 160
  5. If the computed Start Lux and End Lux values need to be adjusted, enter the applicable values in the table or drag the vertical lines on the graph to adjust the Start Lux and End Lux values.
    Trigger values must abide by the following restrictions:
    Trigger[n].start < Trigger[n].end
Trigger[n].end < Trigger[n+1].start
