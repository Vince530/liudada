Tune EV compensation

EV settings specify the standard brightness adjustment of an image.

A setting of EV+1 means the image is two times as bright. Due to complex and nonlinear calculations for luma and luma target, sometimes applying strict standard adjustments causes general images to be too dark or too bright. Adjust the standard EV compensation table for different projects.

By default, the entries increase monotonically and linearly.

Negative EV

  1. If an image is too dark under negative EV settings using the standard EV compensation:
    • Scale all entry values that are less than 1, or
    • Increase a single value with the EV setting of concern and ensure the array still increases monotonically.
    If an image is too bright, decrease the entries the same way.

Positive EV

  1. If an image is too bright under positive EV settings using the standard EV compensation:
    • Scale all entry values that are greater than 1, or
    • Decrease a single value with the EV setting of concern and ensure the array still increases monotonically.
    Do not modify entries that have a value of 1.
