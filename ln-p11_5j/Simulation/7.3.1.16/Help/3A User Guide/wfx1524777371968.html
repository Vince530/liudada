Tune torch flash

  1. Click the AWB tab, then click the Tree View tab.
  2. To enable/disable torch flash, select LED Torch (V1) > Enable and set as appropriate.
  3. To modify the camera type, select LED Torch (V1) > Camera Type and set as appropriate.
  4. To add the coordinates of the LED reference point, select LED Torch (V1) > Illu Spec Refpoint and enter the values in the Rg and Bg fields.

  5. To modify the LED light convergence speed, select LED Torch (V1) > Convergence Speed and enter the appropriate value.
  6. To set a distance weight trigger, do the following:
    1. Select LED Torch (V1) > Illu Spec WV > Trigger > [n] > Trigger Pt, then update the Start and End values.
    2. Select LED Torch (V1) > Illu Spec WV > Trigger > [n] > Trigger > Data, then update the weight value for this distance trigger.
    3. To add more triggers, select LED Torch (V1) > Illu Spec WV > Trigger and click the green plus sign to add a new blank trigger row. Repeat the previous two substeps to set the trigger range and associated weight value.
  7. To set a lux index-based weight adjustment, do the following:
    1. Select LED Torch (V1) > Illu Spec WVInterp > Trigger > [n] > Trigger Pt, then update the Start and End values.

    2. Select LED Torch (V1) > Illu Spec WVInterp > Trigger > [n] > Trigger > Data, then update the interpolation ratio for this trigger.
    3. To add more triggers, select LED Torch (V1) > Illu Spec WVInterp > Trigger and click the green plus sign to add a new blank trigger row. Repeat the previous two substeps to set the trigger range and associated interpolation ratio.

  8. Click File > Save Project.
