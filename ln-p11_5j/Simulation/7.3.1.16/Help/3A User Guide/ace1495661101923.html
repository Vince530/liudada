Adjust ROI parameters

Use the AF Tuning tab in the Chromatix tool to make adjustments to the ROI parameters.

  1. In the Chromatix tool, select File > Open Project to open the project you want to tune.
  2. Click the AF Tuning tab.
  3. Expand the Miscellaneous AF Tuning section of the screen. The ROI Tuning group displays the active values for each ROI parameter.
  4. Do one of the following:

    • Enter a new value to change a specific parameter. The Focus Emphasis indicator changes to Custom values.

    • Select a preset from the Focus Emphasis list. The parameter values change according to the selected preset. The following table describes each preset.

      Focus Emphasis Description
      Robustness
      • Designed for those that prefer a more robust focus window.
      • Larger ROI size
      • Focus value will have more details in the ROI
      • If there are multiple objects in the ROI, multiple peaks are common
      Small Objects
      • Designed for those that prefer a focus window for small objects
      • Smaller ROI
      • Less detail
      • Focus value curve will be very flat
  5. Select Original values to return to the defaults.
