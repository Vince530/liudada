Tune touch AEC

Touch AEC modifies the luma metering weight based on the specific region touched on the preview display screen.

  1. In the Parameter Editor, navigate to 3A > AEC > Touch > Touch Mtr.
  2. To enable Touch ROI, select the Value check box next to Enable Touch ROI.
  3. For Weight Touch ROI, enter a weight value between 0 and 1. The metering weight determines the effect that the touch area has on the overall frame luma. Enter a higher value to apply more weight on the touch region.
    • 0 − No influence from selected touch UI area
    • 1 − Spot meter on the selected UI area
    • 0.5 − Both the selected touch are and full frame FOV are averaged
    • The default value for touch Weight is 0.8.
  4. Verify touch AEC tuning by pointing the camera to a chart similar to the following image and tapping the touchscreen. Repeat for all four corners and the center rectangle.
  5. To enable dynamic ROI AEC tuning, select the Value check box next to Enable Touch ROI Hist Adj.
    • Dark ROI hist adjustments
    • Bright ROI hist adjustments
    1. For brighter ROI to be darker, increase the end value for darkLumaRatioRange. For dark ROI to remain dark, decrease the start value for darkLumaRatioRange.
    2. For brighter ROI to remain bright, increase the end value for brightLumaRatioRange. For darker ROI to be brighter, decrease the value for brightLumaRatioRange
