Outlier distance parameters for AWB dual camera sync

The outlier distance parameters are used to set the GCM region to collect GCM stats.



It is recommended to use the default value for GCM outlier distance. If adjustment is necessary, determine the adjustment value with a gray chart image slightly based on Lux ID and CCT. To minimize artifacts from module-to-module deviation, set with enough distance margin.
