Stats screening concept

Very dark stats and very bright (saturated) stats, can adversely impact Bayer grid stat information, such as R/G and B/G stat average values. Very dark stats can be contaminated by several noise sources, while highly saturated nongray stats can be treated as gray stats. This can cause misleading color stats.

The stats screening process eliminates dark/saturated stats that can effect the final Bayer grid stat outcome. The following example shows stats taken from a 2x2 Bayer stat (R, B, Gr, Gb). If the R, G, B stat is lower than a specified threshold, i.e., the dark pixel threshold value, the stats screening process can filter out this stat. Conversely, if the R, G, B stat is higher than a specified threshold, i.e., the saturated pixel threshold value, the stats screening process can filter out this stat.
