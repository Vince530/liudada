Fine-tune dynamic skip frames

Use this procedure to set up a series of triggers to skip a defined number of frames based on the difference between current and previous frames.
  1. Open the project and click the AWB tab.
  2. On the List View tab, select Dynamic Skip Frame.

  3. To edit the difference ratio trigger range, in the first column, double-click the entry in brackets. Update the start and end values and click outside the box to save the change.
  4. To edit the number of frames to skip core data, in the second column, double-click the entry in brackets. Update the value for this trigger and click outside the box to save the change.
  5. To add more trigger levels, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous three steps to set the trigger range and associated core data.
  6. Click File > Save To Project to save the changes.
