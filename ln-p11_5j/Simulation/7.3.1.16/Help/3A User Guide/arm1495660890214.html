Tune the exposure table

The AEC exposure table controls image brightness through combinations of sensor exposure times and sensor analog gains. When creating a new tuning project, the sensor driver info file is used to automatically create the exposure table using the values provided by the sensor vendor. However, these values can be modified using this procedure.

Per Scenario values are taken from any 3A defaults provided in Chromatix.

The Max Analog Gain value comes from the sensor data and cannot be changed.

Note: When changing between modes (e.g., Snapshot and Video), the exposure table must be tuned.
  1. On the AEC tab, click Exposure Table under the List View.
    The first exposure table shown is created based on defaults within Chromatix.
    Note: Tree View is for advanced users only.
  2. To add another exposure table, click Add Table.
    1. Click Add to add as many additional exposure table rows (knee points) as needed.
    2. Adjust Gain and Exp Time (ms) to adjust the exposure index.
    3. Select the increment priority for every knee point under IncrementPriority. The final knee point is always set to Gain.
    4. In the Table Usage group, select the table number that applies for Snapshot, Pre Flash, Main Flash, Flash Measurement, Aggresive, and Fast AEC.
      e.g., Snapshot may use table 0, but Pre Flash may use table 1.
  3. Check the gain and exposure time values as shown in the Exposure Plot.
    The plot is multi-axis. To read the gain, use the left axis (Gain); to read the exposure time, use the right axis (Exposure Time (ms).

    The increments on the plot will adjust based on the chosen increment priority.

  4. Adjust Histogram values.
    The Enable Hist Metering, Enable ADRC, and Enable Dark Boost selections and triggers are based on defaults within Chromatix.
    1. Double-click a trigger to edit the start and end values.
    2. Click or to add a trigger above or below the currently selected trigger, respectively.
    3. To delete a trigger, select it and click .
    4. To copy one trigger value to another, select one trigger and click . To paste, select another trigger and click .
  5. Double-click any parameter in the Data column to edit the values.
    The parameters shown in the histogram plot are Sat Ref Low, Sat Ref High, Dark Ref Low, and Dark Ref High. The range for these parameters is [0,255].
  6. Select the plot data source.
    Choice Description
    Project Plots data from the project only.
    Image Plots data from any JPEG loaded only.
    Both Plots data from both the project and any JPEG loaded.
    Black data points shown on the plot are taken from any loaded image and are not adjustable; red data points are taken from the current project and are adjustable.
