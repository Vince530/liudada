Tune ISO value

ISO is a measurement of the sensitivity of traditional photographic film. For digital cameras, the term is also used as a measurement of the sensitivity camera sensor. ISO 100 gain is optimum for digital cameras.
The goal of this procedure is to calibrate the QTI ISO Value so that it equals 100 (same as optimum sensitivity = ISO 100). The calibration tool computes an ISO 100 Gain, which is then used to update the QTI ISO Value.
  • QTI ISO Value begins as real gain (from the Exposure Table)
  • Using a RAW image captured with the tuning device, use the AEC dialog to compute the gain associated with ISO 100. This is referred to as the Preview ISO 100 gain.
  • If real gain is > ISO 100 Gain, the QTI ISO Value is recalibrated with (Real Gain * 100/ISO 100 Gain).

In ISO 2721:2013 (https://www.iso.org/obp/ui/#iso:std:iso:2721:ed-2:v1:en), a lookup table determines the relationship between ISO value and "standard" ISO value. Starting with 3A v5.1, standard ISO value is available by enabling the ISO quantization function to convert the ISO value to standard ISO value.

  1. In a light box, capture raw image of the MCC chart using brightness conditions between 700 and 1000 lux with D65 color temperature.
  2. Verify that the brightest patch is not saturated. If the patch is saturated, capture a wider field image or repeat luma target and tolerance tuning using a decreased luma target value.
  3. Using a light meter, measure the luminance (lux) shining onto the surface of the white patch (#19) of the MCC chart.
  4. Navigate to the AEC tab and enter values as needed.
  5. Verify that the ISO 100 Gain value is correctly set by capturing the analog gain value in a light source box. The ISO value in the EXIF can be verified using the following formula: ISO value = (analog_gain*100)/ISO_100_gain
