Reduce flickering during AE convergence

  1. Verify with the sensor vendor that the correct sensor gain and line count (exposure time) setting has been implemented in the sensor driver. Usually, the group parameter hold register is used so that gain and line count (exposure) can be synchronized.
  2. Read the I2C and dump the kernel log for the AEC registers.
  3. Follow the guidelines provided by the sensor vendor to implement the sensor driver for AEC tuning. Verify the camera AE after making any adjustments suggested by the vendor.
  4. If the issue persists, use the exposure table to verify that the step between each exp_index value is close to 3%. Flickering can occur if the step is too far away from 3%. Tune the exposure table to reduce flickering.
  5. Capture an image with the test condition to verify the results.
