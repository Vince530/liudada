LED measurement

LED measurement tuning controls the parameters that compute flux, and r/g and b/g ratios during flash tuning and calibration. LED measurement applies to LED tuning and LED calibration.

Parameter Description Tuning Default
frameErrAllowance Increase this parameter to allow more frame-to-frame differences in the average r/g and b/g color ratios. Range: [0.005, 0.03] 0.015
numOfFrames Specifies the number of frames to average r/g and b/g color ratios for a point. Increase this value to get more robust calculations. Range: [3, 5] 3
maxNumOfFrames Increase this value to allow the algorithm to try more if the frameErrAllowance parameter is failing. Range: [6, 10] 7
maxNumOfExtraAttempts If measurement error exceeds the frameErrAllowance parameter threshold, the measurement is considered to be failed; however, further tries are allowed. This parameter controls how many extra tries total attempted to make the requested measurements. Range: [2, 10] 3
