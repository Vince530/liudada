Increase image brightness in gray chart and resolution chart scenes

Gamma also impacts the final brightness, so tuning the gamma curve must be done before tuning the AE luma target.

Recommendation: Tune the luma target for both the gray chart and resolution chart scenes.



  1. Capture test images with 3A EXIF information and AEC log messages.
  2. Select File > Open Image and open the images in the Image Viewer.
  3. In the 3A Debug dialog box, click the Data tab and select AEC Debug Info > Convergence Information > . Note the Lux index value.
  4. On the Data tab select AEC Debug Info > Luma and Luma Target Information. Ensure the Bayer Metered Luma (frame luma) is almost equal to the AE target before continuing with the next step.
  5. Tune AE luma target and tolerance.
  6. On the AEC Fine Tuning tab, open the Luma Target Tuning panel. Compare the Luma Target Tuning panel configuration to the Lux Index to see which Luma Target is used in the test condition (for example: Zone 0, Zone 1, Zone 2).

    For example, if the Lux Index is 318 then Trigger 0 End < Lux Index < Trigger 1 Start, so the Luma Target (50) of Zone 1 is used for this image. If the Lux Index is within a trigger, e.g., Lux Index = 397, the luma target value is a interpolated value between luma targets of Zone 1 and Zone 2. The real number used in the algorithm is 45.

    Real_luma_target = luma_target_zone_1 + (luma_target_zone_2 – luma_target_zone_1) * [(lux_idx – trigger1_start) / (trigger1_end – trigger1_start)].

  7. On the AEC Fine Tuning tab, click Edit Header. Select Header type: 3A and navigate to AEC > Luma Targets > Luma Target Data.
  8. Enter the Luma Target in the appropriate Region.

  9. Capture these images in the resulting test condition:

    • Flat gray surface in a light source box (LSBD)

    • Image of the ISO resolution chart

    A side effect may be the image is overexposed if the luma target is too large. Tuning engineers can repeat this procedure to get the desired results.
