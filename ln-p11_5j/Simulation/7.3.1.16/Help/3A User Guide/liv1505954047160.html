Custom reference point parameters

To view the customer reference point parameters, open a project and click Custom Reference Point in the Tree View tab.

Table : Custom reference point tuning parameters
Parameter name Description Tuning Factory default
Enable Flag

Enables or disables each custom reference point

  • 1 – Enable
  • 0 – Disable
 Set to 1 to use custom reference points. 0
Ref Pt
  • Defines the reference point for the custom illuminant
  • Specified by the vertex of the R/G and B/G ratios (R/G, B/G)
  • Add as many custom reference points as needed by entering the R/G and B/G values.
  • Measure the R/G and B/G values with a gray chart under the custom illuminant.
 (0, 0)
Dist WV: Trigger Pt Structure that defines distance zones to which weight values are assigned. The distance is the measurement between the custom reference point and an input stat. Set up one or more distance zones by specifying the start and end distance for each zone (e.g., Zone1 = 0 to 0.5, Zone2 = 0.9 to 1.0).

(0, 0)

No predefined zones are created during AWB initial tuning.
Dist WV: Trigger (Core Data) Defines the weight value assigned to each of the distance zones. For each distance zone, specify the weight value to assign to that zone. 0
Weight Adj: Trigger Pt Structure that defines lux index zones to which weight adjustment values are assigned. The lux index comes from the image data. Set up one or more lux index zones by specifying the start and end value for each zone (e.g., Zone1 = 0 to 280, Zone2 = 310 to 360).

(0, 0)

No predefined zones are created during AWB initial tuning.
Weight Adj: Trigger (Core Data) Weight adjustment is a factor multiplied with the distance weight. This adjustment factor allows flexible control of enabling and disabling customer reference points based on a given lux index. For each lux index zone, specify the weight adjustment factor to assign to that zone. 0
