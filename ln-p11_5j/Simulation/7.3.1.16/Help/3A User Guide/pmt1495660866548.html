Tune AF step size manually

The AF Tuning tab calculates the CAF/TAF step tables based on values you specify in the Lens Position Tuning Input group. You can adjust the tables by selecting one of the Search Preference presets or by entering new values.

Use the following guidelines when determining custom step size values:

  • The algorithm step unit is the same as DAC. This means that one step is equal to one DAC.
  • The lens shift for macro and far objects is different. In the macro range, the step size can be increased while maintaining overlapping DOF. If steps are set too large, the DOF may miss some object distance and the actuator may not be able to handle the move.
  • The step size table has five regions, separated by four boundaries. The first boundary is set as a hyperfocal position and the remaining boundaries are configurable. By default, the boundaries are set to 50 cm, 20 cm, and 14 cm due to the nonlinear characteristics. If needed, adjust the boundaries in the Chromatix header editor using the 3A > AutoFocus > Tuning Algorithm > Single > Optics parameters.


  • There are five regions defined in the lens movable range:

    • Region 0: From far_end_position to normal_hyperfocal _position

    • Region 1: From normal_hyperfocal _position to srch_rgn_1 (default = 50 cm)

    • Region 2: From srch_rgn_1 to srch_rgn_2 (default = 20 cm)

    • Region 3: From srch_rgn_2 to srch_rgn_3 (default = 14 cm)

    • Region 4: From srch_rgn_3 to near_end position

  • A large step size can improve the AF speed, but can cause the FV curve to miss the peak for that DOF. Generally, the macro region has a larger step size and the infinity region has a smaller step size.
  • In general, the normal light region has a smaller step size and the low light region has a larger step size. In low light, the frame rate is dropped and, in order to maintain AF speed, the step size is increased slightly compared to normal light.

    Prerequisite: Before tuning AF step size, ensure that the actuator damping is well-tuned. Also ensure that the actuator can focus to infinity (typically ~3 m) and to macro (typically 10 cm) distance, as specified by the vendor.
