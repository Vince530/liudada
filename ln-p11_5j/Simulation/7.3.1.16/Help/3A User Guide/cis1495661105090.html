LED smart ROI flash estimation parameters

In AEC flash estimation, the smart ROI estimation algorithm can detect close objects within the scene.

To tune the parameters, in the Parameter Editor, navigate to 3A > AEC > Flash > Flash Mtr > Smart Flash.

Parameter Description Tuning Default
smart_flash_est_enable

Use this flag to enable the smart ROI estimation algorithm for AEC.

Enable smart ROI estimation to achieve better saturation prevention for close foreground objects.

0 (disable)

1 (enable)

 1 (enabled)
smart_flash_est_strength

This value controls how much AEC ROI estimation should focus on the foreground objects.

This parameter might be useful for fine tuning certain scenes. A setting of 0.0 essentially disables the smart ROI estimation for AEC.

 Range: [0.0, 2.0]

Higher value = focus more on foreground

Lower value = focus more on background brightness

 1.0
minOffStats Increase this value if the image sensor is noisy in dark scenes, causing large exposure variations from shot to shot Range: [2, 5] 3
kFloorPct Increase this value to emphasize small reflective objects and prevent saturation. Range: [0.0, 0.95] 0.8
satPreventStrength Increase this value to further reduce saturation. Range: [0.0, 1.0] 0.0
enableSpeckleHandling Enable this feature to detect speckle corner cases where a reflective surface (e.g., a mirror) produces a saturated speckle that would otherwise cause underexposure. This feature is somewhat opposite to saturation prevention as it allows saturation for speckles.

0 (disable)

1 (enable)

 1
speckleSaPctRange Decrease this range to allow speckle detection to kick in more. Range: [0.01, 1.0] [0.05, 0.35]
