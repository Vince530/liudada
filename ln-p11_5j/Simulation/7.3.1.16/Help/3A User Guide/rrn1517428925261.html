Dual camera AEC

Identical image brightness is preferred for seamless transition or seamless composition of multiple cameras. For AEC, this means an identical exposure to light. Applying the same exposure gain and exposure time is not sufficient for matching exposition (e.g., if lenses, sensors, Bayer filter, etc. are different). The sensitivity of color filters diverges under different ambient light color temperatures. Under illuminants (A, D65, etc.) the sensitivity difference diverges. This requires CCT-based fine adjustments.

Figure : (Above) before dual AEC tuning, (Below) after dual AEC tuning

Calibration is performed by the module vendor. The module vendor will also program the camera module OTP data. Camera software loads calibration OTP data if available. Calibration replaces tuning Dual Cam Exp Comp Multiplier as the CCT-based adjustments are not calibrated so the tuning values remain the same.

When comparing jpeg images, there may be some differences in brightness. This is caused by each camera computing the lux index independently. Many ISP blocks, including tone reproduction blocks that affect brightness use lux as a trigger.
