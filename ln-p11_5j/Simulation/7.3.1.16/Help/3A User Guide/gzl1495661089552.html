Initial tuning procedure for unidirectional actuator

Repeat the following steps for each mode (snapshot, display, video, etc.) in the tuning project. All of the parameter and parameter structures in this section are located at

  1. Copy the lens postion references from the worksheet to the Index located at 3 A > AF > Chromatix Contrast AF > Single Search in the Parameter Editor.
  2. Click the AF Tuning tab.
  3. In the Lens Position Tuning Input group, clear the Bi-Directional check box.
  4. Enter the appropriate values in the Lens Position Tuning Input group and click Calculate. Based on the information you enter, the tool calculates and displays the initial set of tuning values used for lens positions, TAF steps, CAF steps, and SAD parameters.
  5. Chromatix also calculates the default values for the parameters in the Optics structure. These parameters control the search behavior, boundaries, and search regions of the single AF algorithm. Click Edit Header, change the header type to 3A, and search for Optics to view these parameters.

    The following table describes the purpose of each parameter and cases where the defaults require adjustment.

    Parameters Search behavior Description
    • CAF Far End
    • CAF Near End
    • TAF Far End
    • TAF Near End
    		 Search boundary

    Typically, search boundaries are set as same near end and far end for the opened lens position range. However, because of defocus and other limitations, reduce the search range to avoid search failures. In this case, adjust the range by adjusting the CAF Far End and CAF Near End for CAF mode, and TAF Near End and TAF Far End for TAF mode.

    For video, the CAF Near End is set to 14 cm by default because of the limitation of defocus associated with small step size.
    • Search Region 1
    • Search Region 2
    • Search Region 3
    		 Search regions

    Used for obtaining step size. When selecting the step size Search Region 1, Search Region 2, and Search Region 3 are looked up for the corresponding entry. When adjusted correctly with the step table, a flexible search behavior can be created. By default, the region boundaries are set to hyper focal, 50 cm, 20 cm, and 14 cm for mapping. If the module lens has a large aperture, the region settings can be shifted by one index entry separately.
    Fine Search Region Fine search boundary Fine search improves the accuracy of the images, but it costs extra frames for this search. Therefore, to balance the trade-off, typically theFine Search Region is set to the value of the SINGLE_50CM_INDEX as an initial tuning value. For speed improvement, the boundary can be increased to reduce using fine search. For accuracy improvement, expand the fine search range down toward the macro objects distances. To disable fine search, set it to Infinity index.
    • Far Zone
    • Near Zone
    		 Near and far zone Used to limit the chance of defocus at both ends of the range. If defocus is causing AF failure close to near and far end of the search, reduce the far zone boundary (or increase for near zone) to increase success rate.
    Initial Position Initial position This parameter specifies where the lens position is upon the first AF search in camera start-up. Set to hyperfocal by default. However, to accommodate lens sag in some open-loop actuators, it might have to move back to the infinity limit position.
    Mid Zone Mid zone This parameter defines the starting direction of the lens because the lens moves towards the mid zone.
