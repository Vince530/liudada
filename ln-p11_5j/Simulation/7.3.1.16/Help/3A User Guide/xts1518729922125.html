Tune Dual Cam Exp Comp Multiplier

Tuning the compensation factor, Dual Cam Exp Comp Multiplier, helps align the exposures of multiple cameras.

Capture Dual Cam Exp Comp test images

  1. Set up a light box for D50 and capture one image per camera.
  2. Use a flat, uniform gray chart and set the camera optical axis perpendicular to it. Ensure the gray chart occupies 100% FOV for both cameras.
  3. Open the images in Chromatix 7 and verify Average Luma in the AEC debug information.
    Each image should have an average luma value between 45 and 55.

    If the value does not fall within the recommended range, verify the metering tuning settings and verify that no user interface settings (e.g., manual exposure, EV, etc.) are interfering with the image.
