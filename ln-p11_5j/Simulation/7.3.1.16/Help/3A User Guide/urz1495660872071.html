Tune spotlight conditions

A spotlight condition is when a very bright light, or over exposed object, in a dark area of an image causes a false peak in the focus value (FV). Unless this condition is accounted for, it can result in an out-of-focus or blurry image. The following images show the same scene: one without spotlight (left) and one with spotlight (right).



The following graph illustrates a false peak like one encountered in a spotlight condition. In this example, the FV peak is at the 1 m position, but the best focus is actually at a different position with a lower focus value.



The process of tuning to correct spotlight issues has two basic parts:

  1. Prevent false peaks by adjusting the AF parameters used for low-light conditions.
  2. If more adjustments are needed, turn on spotlight detection assisted AF.
