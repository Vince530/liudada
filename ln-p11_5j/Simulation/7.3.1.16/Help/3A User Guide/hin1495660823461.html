Distance to lens mapping

This test checks the lens position tuning values calculated in theAF Tuning tab with the lens positions captured in a full-sweep test. If the lens position mapping does not match the test results, then tune the lens positions tuning values. These values are in the AF Tuning tab as well as the single_optic_tstructure in the 3A header.

Setup

Place the ISO 12233 resolution chart 14 cm from the device on a tripod with a 500+ lux light condition. Repeat the test at 20 cm, 60 cm, and 200 cm.

Analysis and corrective action



If the graph looks like Graph A, then the step table is correctly configured. If the graph looks like Graph B, then there is too much deviation between the lens postion measured in the test and the lens postion in the header. Return to the AF Tuning tab and check the Input values. If you update the values, click Calculateto recalculate the lens position tuning values then repeat the test. If the Input values are accurate, adjust the appropriate lens position tuning value then repeat the test.
