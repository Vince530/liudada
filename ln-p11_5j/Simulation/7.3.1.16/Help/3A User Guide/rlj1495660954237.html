Tune flash triggers

Flash triggers control the lux level at which flash will fire when used in auto mode. See LED Flash for more information.

  1. Prerequisite: Ensure that the battery is fully charged before performing this procedure. As described in PM8952/PM8956 + PMI8952 Power Management IC Design Guidelines/Training Slides (80-NT390-5), flash current is derated beyond a certain threshold at lower battery levels, resulting in lower intensity flash and thus degraded image quality. Work with the QTI PMIC team to understand the device behavior, assess the image quality impact, and determine whether to allow flash below a certain battery level.
  2. Enable the 3A debug log.
  3. Point the camera at the scene where the LED flash is to be used. Note the value of the Lux Index parameter from the 3A debug log.
  4. In the Parameter Editor, navigaet to 3A > AEC > Flash Ext > Flash Lux Trigger.
  5. Enter the Lux Index start and end values as noted in step 3 from the debug log.
  6. If necessary, adjust the value of the flashLuxTrigger start and flashLuxTrigger end parameters to trigger auto flash based on scene brightness:

    • Increase the value to trigger LED flash in darker scenes.

    • Decrease the value to trigger LED flash in brighter scenes.

    It is recommended to separate the flashLuxTrigger start and flashLuxTrigger end parameter values by a minimum of 10 lx. The flashLuxTrigger start value must always be a smaller than the flashLuxTrigger end value.
