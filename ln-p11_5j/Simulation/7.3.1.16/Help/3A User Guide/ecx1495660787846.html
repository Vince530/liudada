Tune histogram AEC luma target

The histogram AEC luma target adjustment feature modifies the luma target based on histogram analysis.

Standard AEC tuning features primarily work by changing the luma calculation. This can result in stability issues if parameters are overtuned (e.g., bright region, dark region). Tuning the luma target based on histogram analysis results in better exposure for a variety of scenarios. Minimal tuning is needed to achieve reasonable effects. Histogram AEC is not heavily tied to sensor characteristics, so retuning is not necessary for different sensors.
The histogram AEC luma target feature can be effective in the following situations:
  • Improves brightness compensation for a flat scene (e.g., white board). Histogram AEC can be used instead of the flat scene detection feature.
  • Increases brightness in certain low dynamic range scenes to improve visibility in darker areas and overall contrast.
    Figure : (L) Histogram metering applied; (R) Original image

    The image on the left shows the histogram AEC algorithm applied. The feature increases the exposure to fill the histogram range, up to a cap, which improves contrast and dark area visibility.

  • Prevents mild saturation in high dynamic range scenes that have overexposed areas.

The histogram AEC feature is is tuned using the generic multizone tuning guidelines. Default AEC triggers are initially tuned in a common step and then copied over to histogram metering.

max_target_adjust_ratio
  • The maximum adjustment that can be made by the algorithm. This parameter is multizone enabled.
  • The ratio is computed as the maximum luma target over the current luma target. For example, if the current luma target is 40 (after lux interpolation) and Max Adjust Ratio is 1.8, then max luma target is 40*1.8 = 72
  • For low light (Zone 2), it is possible to allow positive target adjustment, but it is stongly recommended not to by keeping this ratio set to 1.0.

min_target_adjust_ratio
  • The minimum adjustment that can be made by the algorithm. This parameter is multizone enabled.
  • The ratio is computed as the minimum luma target over the current luma target.
0.8 to 1.0 1.0
target_filter_factor
  • Use this parameter as needed to control the temporal filter on the adjusted luma target for stability purposes.
  • Increase the value for more stability and decrease the value for more swift response. The higher the value, the stronger the filtering, and the more stable the transition during panning. The side effect of high filtering is that convergence can be slower in certain quick panning cases.
 0 to 0.99 0.5
hist_sat_pct (0x308 header)
  • The percentage of pixels used to calculate the saturation position.
  • No tuning is required for this parameter. Do not change this value unless you fully understand the algorithm. The only benefit to tuning this parameter is to improve stability by increasing it.
 0.1 to 0.2 0.15
hist_sat_pushup_range (0x309 header)
  • The saturated percentage range for push-up compensation (flat/low dynamic range scenes), such as when luma target is positively offset.
  • It is recommended to use a bigger range (e.g., 0 to 0.08) to keep AEC stable and avoid sudden changes of brightness when panning in and out of a flat scene.
  • Allows specification from the top start to top end percent of pixels. Allows for ignoring the first start percent of pixels for the calculation.
  • Allows for defining the range for each R/G/B channel in triggered brightness. zones.
  0.02 (start)

0.08 (end)
hist_sat_pushdown_range (0x309 header)
  • The saturated percentage range for push-down compensation (saturation prevention), such as when luma target is negatively offset.
  • It is recommended to use a smaller range (e.g., 0.01 to 0.03) to allow AEC to effectively respond to small saturation within the scene. Starting from a value larger than 0 also allows AEC to ignore the first few saturated pixels.
  • Allows specification from the top start to top end percent of pixels. Allows for ignoring the first Start percent of pixels for the calculation.
  • Allows for defining the range for each R/G/B channel in triggered brightness zones.
  0.01 (start)

0.04 (end)

hist_sat_low_ref
  • The lower end of the control range for the calculated saturation position.
  • The higher the low reference point, the more aggressively the algorithm pushes the luma target up in flat scene and low dynamic range scenes.
  • The algorithm attempts to adjust the luma target so that the saturation position is within the range specified. The reference points are in the linear domain; consider the gamma effect that is applied to these two parameters.
  • The narrower the range between the low and high reference points, the more likely that the luma target is adjusted to shift the saturation position.
  • There is no guarantee that the saturation position, after AEC converges, falls within the specified range for all scenes because there are other constraints, such as darkness control range and the max/min target adjustment ratio.

  • There is no guarantee that the saturation position, after AEC converges, falls within the specified range for all scenes because there are other constraints, such as darkness control range and the max/min target adjustment ratio.

hist_dark_low_ref
  • The lower end of the control range for the calculated darkness position.
  • Only reduce this value if it is needed to prevent saturation in a high dynamic range scene with a lot of dark areas. In all other cases, do not change this value.
  • The algorithm attempts to adjust the luma target so that the darkness position is within the range specified. The reference points are in the linear domain; consider the gamma effect that is applied to these two parameters.
  • This pair of points can have a much larger range between them. If the range is too narrow, there could be stability concerns. In addition, it is generally not necessary to have very precise control of the dark area percentage.
  • There is no guarantee that the saturation position, after AEC converges, falls within the specified range for all scenes because there are other constraints, such as saturation control range and the max/min target adjustment ratio.

  1. In the Parameter Editor, navigate to 3A > AEC > Metering > Mtr Luma Target > Hist Luma Target.
  2. Click Enable Hist Metering and select the Value check box.
  3. Ensure that the Enable check box is selected and verify the results achieved from enabling the default parameter values. Adjust individual parameters if necessary to achieve more exact brightness matching scene-by-scene, or to improve stability.
  4. If individual parameter adjustments are necessary, click Hist Luma Target Zone and add zones using .
  5. Adjust the Zone Trigger values for each zone.
  6. Adjust the Zone Data parameter values for each zone.
    Parameter name Description
    Min Target Adj Ratio The minimum adjustment that can be made by the algorithm. The ratio is the minimum luma target over the current luma target. The default value is 0.6.
    Max Target Adj Ratio The maximum adjustment that can be made by the algorithm. The ratio is the maximum luma target over the current luma target. The default value is 1.0.
    Sat Ref Low The lower end of the control range for the calculated saturation position. The default value is 100.
    Sat Ref High The higher end of the control range for the calculated saturation position. The default value is 100.
    Dark Ref Low The lower end of the control range for the calculated darkness position.
    Dark Ref High The higher end of the control range for the calculated darkness position.
    Hist Sat Pushup Range In each zone, for each color channel (R/G/B), enter a Start and End value to define the saturated percentage range that gets push-up compensation (flat/low dynamic range scenes), such as when luma target is positively offset.
    Hist Sat Pushdown Range In each zone, for each color channel (R/G/B), enter a Start and End value to define the saturated percentage range that gets push-down compensation (saturation prevention), such as when luma target is negatively offset.
    Hist Dark Range In each zone, for each color channel (R/G/B), enter a Start and End value to define the range of pixels used to calculate the darkness position.
    Max DRC Gain The max ratio between long and short exposure.
  7. (Optional) Click Target Filter Factor and set the value, if needed.
  8. (Optional) Click Enable Sat Pos Clamping and select the Value check box if clamping the saturation position calculation with AWB gains is needed.
