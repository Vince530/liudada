Tune ISO quantization

ISO values from a camera originate from film photography, and were used to measure the speed of films (i.e., how fast they can expose and form an image). Films are rated with standardized ISO values (such as 100, 200, 400, and 800) (see the ISO_12232 standard for more details).With digital cameras, those standardized values are inherited; however, digital camera ISO values can be calculated directly by converting the total gain. Hence, the values are continuous and not quantized to standardized values, for example, 465, 629, 1215 etc. From AEC v6.0 onwards, there is the option to quantize these values to standardize them to ones defined in the ISO 12232 standard.

  1. In the Parameter Editor navigate to 3A > AEC > Arbitration > ISO Data.
  2. To enable ISO quantization, select the Value check box.
  3. (Optional) Click next to Thres Quantization ISO Value and enter values, if needed.
  4. (Optional) Click next to Quantization ISO Value and enter values, if needed.
