ADB commands

Root access is required to run these adb commands.

ADB commands

Purpose Command
Gain root access 
adb root
adb wait-for-devices
adb remount
adb devices
Enable full sweep/disable algorithm

To perform full sweep from far to near end:

adb shell setprop debug.camera.af_fullsweep 1

To perform full sweep from far to near then back to the far end:

adb shell setprop debug.camera.af_fullsweep 3
Automation/control events

To wake up the camera:

adb shell input keyevent 26
adb shell input keyevent 82

To close the camera:

adb shell input keyevent 4

To issue a tap on screen with coord. (X, Y):

adb shell input tap X Y

To take a picture:

adb shell input keyevent 27
